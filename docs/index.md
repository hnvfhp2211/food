##  <span style="font-family:楷体; font-size: 20px; ">附Markdown常用命令和项目结构如下：</span>


### 常用命令

* `mkdocs new [dir-name]` - 新建项目.
* `mkdocs serve` - 启动本地实时预览服务.
* `mkdocs build` - 构建静态网站，生成可部署文件.
* `mkdocs -h` - 查看帮助信息.

### 项目结构

    mkdocs.yml    # 项目的核心配置文件.
    docs/
        index.md  # 存放所有Markdown文档的文件夹.
        ...       # 网站首页.
