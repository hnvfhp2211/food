### <span style="font-family: 华文行楷; font-size: 30px; color: black;">菜名：麻婆豆腐</center> ###
<span style="font-family: 华文琥珀; font-size: 20px; color:orange;">菜系：川菜    
<span style="font-family: 华文琥珀; font-size: 20px; color:red;">菜式：热菜

---
#### <span style="font-family: 华文行楷; font-size: 30px; color:;">来源：</span> ####
<span style="font-family: 华文行楷; font-size: 20px; color: black;">麻婆豆腐堪称川菜的经典代表，闻名遐迩。它起源于清朝同治年间，由陈麻婆所创。这道菜以嫩豆腐为主料，将豆腐切成均匀的小块，放入加了盐的开水中焯烫，既能去除豆腥味，又能让豆腐更加紧实。
![](https://pics5.baidu.com/feed/503d269759ee3d6d80dee940ffef25334e4ade28.jpeg@f_auto?token=a79c3b1f12cab1d77cae1476036c8a61)
#### <span style="font-family: 华文行楷; font-size: 30px; color: blue;">做法:</span> ####
<span style="font-family: 华文行楷; font-size: 20px; color: black;">烹饪时，锅中倒油，放入剁碎的牛肉末煸炒至变色，加入豆瓣酱炒出红油，此时香辣味瞬间弥漫开来。接着，倒入适量的高汤，放入焯好的豆腐块，轻轻晃动锅子，让豆腐均匀受热，加入适量的盐、生抽、老抽调味。炖煮片刻后，分几次加入水淀粉勾芡，让汤汁浓稠地包裹住豆腐。最后，撒上花椒粉、葱花。麻婆豆腐上桌，色泽红亮，豆腐嫩滑无比，麻辣味厚，肉末酥香，每一口都充满浓郁的香辣气息，让人欲罢不能，充分展现了川菜 “麻辣鲜香烫” 的特点。
### <span style="font-family: 华文行楷; font-size: 30px; color: black;">菜名：回锅肉###
<span style="font-family: 华文琥珀; font-size: 20px; color:orange;">菜系：川菜    
<span style="font-family: 华文琥珀; font-size: 20px; color:red;">菜式：热菜

---
#### <span style="font-family: 华文行楷; font-size: 30px; color:;">来源：</span> ####
<span style="font-family: 华文行楷; font-size: 20px; color: black;">回锅肉被称为 “川菜之首”，是一道家喻户晓的家常菜。它选用肥瘦相间的五花肉，这是保证口感的关键。先将五花肉冷水下锅，加入葱姜、料酒，煮至八成熟，捞出晾凉后切成薄片。
![](https://pics0.baidu.com/feed/962bd40735fae6cd7e7f4712b24a473540a70f80.jpeg@f_auto?token=397f631d944ed760d36a97c80d18a6da)
#### <span style="font-family: 华文行楷; font-size: 30px; color: blue;">做法:</span> ####
<span style="font-family: 华文行楷; font-size: 20px; color: black;">锅中放少许油，放入肉片煸炒，将肉片炒至微微卷曲，出油。此时，肉片变得金黄焦香。接着，加入青椒块、蒜苗段一起煸炒，炒出香味。再放入豆瓣酱、豆豉炒出红油，让每一片肉和蔬菜都裹上红油。根据口味加入适量的盐、白糖、生抽调味。回锅肉色泽红亮，肥而不腻，肉片咸香，青椒和蒜苗增添了清爽的口感，与香辣的酱料完美融合，是一道让人百吃不厌的川菜经典。
### <span style="font-family: 华文行楷; font-size: 30px; color: black;">菜名：宫保鸡丁 ###

<span style="font-family: 华文琥珀; font-size: 20px; color:orange;">菜系：川菜</span>

<span style="font-family: 华文琥珀; font-size: 20px; color:red;">菜式：热菜</span>

---
#### <span style="font-family: 华文行楷; font-size: 30px; color:;">来源：</span> ####
<span style="font-family: 华文行楷; font-size: 20px; color: black;">宫保鸡丁是一道具有独特魅力的川菜，融合了多种味道。它以鸡肉为主料，将鸡肉切成丁，用盐、料酒、淀粉腌制片刻，让鸡肉更加嫩滑。
![](https://pics0.baidu.com/feed/aec379310a55b3194ce6e7d0ff50ca37cefc1706.jpeg@f_auto?token=ae19521700ff729bb7b647e50502708a)
#### <span style="font-family: 华文行楷; font-size: 30px; color: blue;">做法:</span> ####
<span style="font-family: 华文行楷; font-size: 20px; color: black;">锅中倒油，油热后放入腌制好的鸡丁滑炒至变色盛出。锅中留底油，放入花生米炸至酥脆捞出。接着，放入干辣椒、花椒炒出香味，加入葱姜蒜爆香。放入炒过的鸡丁翻炒均匀，倒入由白糖、醋、生抽、料酒、淀粉调制而成的料汁，快速翻炒，让鸡丁均匀裹上料汁。最后，放入炸好的花生米。宫保鸡丁色泽棕红，鸡肉鲜嫩，花生米香脆，甜、辣、酸、鲜诸味交融，是一道将多种口感和味道巧妙结合的川菜佳肴。
###  <span style="font-family: 华文行楷; font-size: 30px; color: black;">菜名：鱼香肉丝 ###
<span style="font-family: 华文琥珀; font-size: 20px; color:orange;">菜系：川菜    
<span style="font-family: 华文琥珀; font-size: 20px; color:red;">菜式：热菜

---
#### <span style="font-family: 华文行楷; font-size: 30px; color:;">来源：</span> ####
<span style="font-family: 华文行楷; font-size: 20px; color: black;">鱼香肉丝是一道经典的川菜，虽无鱼却有鱼香味，这独特的味道令人着迷。它以猪里脊肉丝为主料，将肉丝用盐、料酒、淀粉腌制。配菜则选择木耳、胡萝卜、青椒等切成丝。
![](https://pics4.baidu.com/feed/503d269759ee3d6d444cac71feef25334d4ade88.jpeg@f_auto?token=7d2594af0a7a6e2122ae7b75ece136f6)
#### <span style="font-family: 华文行楷; font-size: 30px; color: blue;">做法:</span> ####
<span style="font-family: 华文行楷; font-size: 20px; color: black;">锅中倒油，放入肉丝滑炒至变色盛出。另起锅，放入豆瓣酱炒出红油，加入葱姜蒜、泡椒末炒香。接着，放入配菜丝翻炒均匀，倒入由白糖、醋、生抽、料酒、淀粉调制的鱼香汁，煮至汤汁浓稠。放入炒过的肉丝，快速翻炒，让肉丝均匀裹上鱼香汁。鱼香肉丝色泽红亮，肉丝软嫩，鱼香味浓郁，酸甜辣兼备，葱姜蒜香突出，是一道极具特色的川菜。
### <span style="font-family: 华文行楷; font-size: 30px; color: black;"> 菜名：水煮鱼 ###
<span style="font-family: 华文琥珀; font-size: 20px; color:orange;">菜系：川菜    
<span style="font-family: 华文琥珀; font-size: 20px; color:red;">菜式：热菜

---
#### <span style="font-family: 华文行楷; font-size: 30px; color:;">来源：</span> ####
<span style="font-family: 华文行楷; font-size: 20px; color: black;">水煮鱼是川菜中一道热辣过瘾的美食。它以草鱼或黑鱼为主料，将鱼肉片成薄片，用盐、料酒、淀粉、蛋清腌制，让鱼片更加嫩滑。鱼骨和鱼头切块备用。
![](https://pics4.baidu.com/feed/b21c8701a18b87d66d3e58eebaf160291d30fdca.jpeg@f_auto?token=204224de55848c4459915b20bd7a2bb8)
#### <span style="font-family: 华文行楷; font-size: 30px; color: blue;">做法:</span> ####
<span style="font-family: 华文行楷; font-size: 20px; color: black;">的清水，大火煮至鱼汤变白。将鱼骨和鱼头捞出放入碗底。接着，将腌制好的鱼片逐片放入锅中，煮至鱼片变色熟透，连汤一起倒入装有鱼骨和鱼头的碗中。在鱼片上撒上花椒粉、干辣椒段、葱花，淋上热油，“呲啦” 一声，香气四溢。水煮鱼鱼片鲜嫩，麻辣味浓，汤汁红亮，是一道让人吃得酣畅淋漓的川菜。
###  <span style="font-family: 华文行楷; font-size: 30px; color: black;">菜名：夫妻肺片 ###
<span style="font-family: 华文琥珀; font-size: 20px; color:orange;">菜系：川菜    
<span style="font-family: 华文琥珀; font-size: 20px; color:red;">菜式：热菜

---
#### <span style="font-family: 华文行楷; font-size: 30px; color:;">来源：</span> ####
<span style="font-family: 华文行楷; font-size: 20px; color: black;">夫妻肺片是一道以牛头皮、牛心、牛舌、牛肚、牛肉为主料的川菜。先将这些食材卤制入味，卤制时加入多种香料，如八角、桂皮、香叶、花椒等，让食材充分吸收香料的味道。
![](https://pics1.baidu.com/feed/6a600c338744ebf8067f7ade65009f3b6259a7cb.jpeg@f_auto?token=b52ac8f69a752e349cef7d05aab1589c)
#### <span style="font-family: 华文行楷; font-size: 30px; color: blue;">做法:</span> ####
<span style="font-family: 华文行楷; font-size: 20px; color: black;">葱花等调料，搅拌均匀，制成红油汁。将红油汁浇在切好的食材上。夫妻肺片色泽美观，质嫩味鲜，麻辣浓香，红油的香辣与食材的醇厚完美结合，是一道深受喜爱的川菜凉菜。### 担担面 ###
担担面是四川的传统面食。它的面条细薄，筋道有嚼劲。先将面条放入沸水中煮熟，捞出沥干水分放入碗中。
![](https://pics3.baidu.com/feed/cc11728b4710b912c81600ca7e04b4129345223d.jpeg@f_auto?token=2faad4fc798897ebc6e62d1e36e4fc42)
<span style="font-family: 华文行楷; font-size: 20px; color: black;">接着，调制臊子，将猪肉末放入锅中煸炒至变色，加入芽菜一起煸炒出香味，制成臊子。然后，调制汤汁，在碗中放入红油、花椒油、生抽、醋、白糖、蒜末、葱花、芝麻等调料，加入适量的面汤搅拌均匀。将臊子浇在面条上，撒上花生碎。担担面面条劲道，臊子香脆，汤汁麻辣鲜香，香气扑鼻，是一道让人回味无穷的川菜面食。
