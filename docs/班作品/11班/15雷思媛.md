# 鱼香肉丝

# 食材准备与处理

‌主料选择‌：首选‌猪里脊肉备与处理‌，口感细嫩，也可用精瘦肉，需顺着纹路切丝.

‌经典配菜‌：传统搭配为‌水发木耳、冬笋丝、葱粒‌，家常版常加胡萝卜、青椒丝配色.

肉丝腌制‌：肉丝加盐、料酒、葱姜水抓匀吸水，再加淀粉上浆，最后淋油封水，腌制 10 分钟防粘连。‌

鱼香汁调配比例
‌核心口味‌：需呈现‌咸、酸、辣、甜、鲜‌五味调和，葱姜蒜香浓郁。
‌通用比例‌：推荐"12345"口诀，即 1 勺淀粉、2 勺生抽、3 勺陈醋、4 勺白糖、5 勺清水调匀。
‌简易配方‌：也可按生抽、陈醋、白糖‌2:2:2‌比例，加少许淀粉和水调制。‌‌
# 烹饪步骤与技巧 #
‌滑炒肉丝‌：热锅凉油，五成油温下肉丝滑散，变色即盛出，保持嫩度.
调制“灵魂鱼香汁”‌
按照“2:2:2”黄金比例：‌2勺生抽、2勺陈醋、2勺白糖‌，加入1勺淀粉、3勺清水搅拌均匀备用。此比例能实现酸甜咸辣鲜五味平衡。‌
炒制底料‌：留底油，下‌泡红辣椒‌（或郫县豆瓣酱）及葱姜蒜末，小火炒出红油和香味。
‌合炒收汁‌：倒入配菜炒断生，回锅肉丝，淋入鱼香汁，‌大火快速翻炒‌至汤汁浓稠裹匀食材即可出锅。‌‌
‌
将肉丝加入‌1勺生抽、1勺料酒、1勺淀粉、1个蛋清‌，抓匀至发黏，再淋入少许食用油封住水分，腌制10–15分钟，使肉质滑嫩不粘连。‌
