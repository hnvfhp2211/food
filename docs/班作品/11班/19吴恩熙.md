# 食材 #
五花肉 500克   
葱、姜 少许    
大料（八角） 2-3颗  
黄酒 100克   
生抽 2汤匙  
醋 半汤匙  
红烧汁 1汤匙  
蚝油 1汤匙  
冰糖 50克  
盐 少许  
鸡精 少许  
草果 一个


# 做法 #
1、五花肉切成块，用清水浸泡洗净。

2、锅中加入清水，放入1勺料酒，1个八角，葱段和姜片各两片，大火烧开，撇去浮沫。

3、把焯过水的五花肉捞出控干。

4、锅中倒入少许油，小火煸五花肉，煸汁微微焦黄。

5、锅中加入2勺生抽、1勺老抽和1勺料酒，炒匀炒香。

6、再加入一听雪碧，如果肉多，可以用清水补足。

7、水量稍稍没过肉为宜。

8、把料包混合放入调料袋中，投入锅内。

9、放入5个干辣椒、2片姜片和冰糖，大火烧开后转小火，盖上盖子，保持小开锅状煮1个小时。

10、1个小时后，放入6个熟鸡蛋。

11、如果不够咸味可以此时加盐调味，我试了一下味道，够咸就没有放盐。

12、盖上盖子继续炖煮半个小时左右，待肉质软烂后，打开盖子，大火收汁即可 红烧肉
