# <center>水煮牛肉 </center> #


## <span style="font-family: 华文琥珀; font-size: 20px; color: black;">来源</span>  ##
‌水煮牛肉起源于北宋时期四川自贡的盐工群体‌，最初是盐工们利用淘汰役牛、就地取材烹制出的一道简朴劳工菜，后经名厨改良成为经典川菜。

这道菜的诞生与自贡千年盐业史密不可分。自贡自古为“盐都”，古代采卤制盐主要依靠牛力驱动辘轳。当牛年老力竭被淘汰后，盐场常将其宰杀，分给盐工作为工钱或食物 。盐工们为补充体力、驱寒祛湿，便将牛肉切片，用盐、花椒、辣椒等加水煮熟食用，这种简单粗犷的吃法被称为“水煮牛肉” 。由于四川气候潮湿，麻辣口味既能刺激食欲，又能暖身御寒，逐渐形成独特风味 。

到了20世纪30年代，自贡名厨‌范吉安‌在传统“渗汤牛肉”（蘸水吃法）基础上进行关键改良：将腌制好的牛肉片直接在辣味红汤中烫熟，一次成味，无需蘸料，使菜品更便捷、风味更融合 。这一创新让水煮牛肉从民间劳工菜走向酒楼食肆，成为盐帮菜的代表作之一 。1981年，水煮牛肉被正式收录进《中国菜谱》，确立了其作为经典川菜的地位 。

如今的水煮牛肉已远非昔日粗食，讲究“麻、辣、鲜、嫩、烫”五味一体，选用牛里脊薄切，辅以豆芽、莴笋等垫底菜，经腌制、炒料、煮肉、泼油等多道工序，最终淋上滚烫红油激香，成就一道红亮油润、滑嫩可口的川味名肴
🍲 ‌水煮牛肉家常做法（1-2人份）
## <span style="font-family: 华文琥珀; font-size: 20px; color: black;">做法</span>  ##

‌
‌食材清单
‌
‌主料‌：牛肉300g（推荐牛里脊或后腿肉，无筋无脂更嫩）
‌配菜‌：豆芽100g、莴笋或白菜适量（传统以豆芽为主）
‌腌肉料‌：鸡蛋清1个、淀粉15g、料酒10ml、蚝油8g、姜丝20g
‌调料‌：豆瓣酱10g、干辣椒粉5g、蒜瓣3瓣、红辣椒1根、香菜5根、花椒适量
‌详细步骤‌
‌处理牛肉‌
牛肉逆纹切薄片（约3毫米厚），加姜丝、鸡蛋清、淀粉、料酒、蚝油抓匀，腌制15分钟。这一步能让肉质滑嫩不柴 。

‌准备配菜‌
豆芽洗净焯水1分钟，捞出铺在碗底；莴笋切片可一同焯熟备用 。
‌炒制红汤‌
热锅凉油，下姜蒜、豆瓣酱小火炒出红油，加入适量开水煮沸，调入盐、生抽、少许糖提鲜。

炒制红汤‌

‌煮牛肉‌
保持汤微沸，将牛肉片‌逐片下锅‌，定型后再轻轻推动，煮约1分钟至变色即捞出，避免久煮变老 。![](https://miaobi-lite.bj.bcebos.com/miaobi/5mao/b%27LV8xNzM1Njc5NDI3Ljc2NTE1NzVfMTczNTY3OTQyOC42ODM3NDc4XzE3MzU2Nzk0MjkuNjU3Ml8xNzM1Njc5NDMwLjM3MzgzMl8xNzM1Njc5NDMxLjEzMzUxNjY%3D%27/4.png)

‌淋油激香‌
将牛肉连汤倒入碗中，表面撒上蒜末、辣椒粉、花椒、香菜段。另起锅烧热油至冒烟，迅速淋在调料上，“滋啦”一声激发出麻辣香气 。

## <span style="font-family: 华文琥珀; font-size: 20px; color: black;">✅ ‌关键技巧‌</span>  ##

牛肉‌逆纹切‌可切断纤维，口感更嫩
腌制时加蛋清和淀粉形成保护层，锁住水分
最后‌热油泼香‌是川菜“点睛之笔”，不可省略
