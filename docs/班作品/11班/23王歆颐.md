# 食材准备 #
‌主料‌：猪里脊肉300克。‌

‌挂糊用料‌：土豆淀粉或玉米淀粉100-150克，常加清水调成糊状，有时加入鸡蛋或少许食用油以增加酥脆

‌‌
‌酸甜汁调料‌：白糖60-100克，白醋或米醋90-100克，少量盐、料酒调味。‌

‌‌
‌配菜与点缀‌：葱丝、姜丝、蒜片、胡萝卜丝、香菜段适量。‌

## 做法 ##
‌处理肉片‌：猪里脊肉切成约3毫米厚的薄片，用刀背轻拍使肉质更嫩，加盐、料酒腌制15-20分钟。‌
‌‌

‌调制面糊‌：淀粉加水调成稠糊，可沉淀后取湿淀粉使用，确保糊能均匀包裹肉片且不下淌。‌

‌‌‌‌
‌油炸肉片‌：
初炸：油温升至160-180℃（约七成热），逐片下入挂糊肉片，炸约3分钟至定型、微黄，捞出。‌
‌‌

复炸：升高油温至180-200℃（约八成热），再次下入肉片快速炸至金黄酥脆，捞出备用。‌

‌‌
‌
‌烹制汁料与翻炒‌：
调汁：将糖、醋、少量酱油或番茄酱（辽宁派风格）混合成酸甜汁。‌

‌‌
炒制：锅留底油，爆香葱姜蒜等配菜，倒入调好的汁熬至浓稠，迅速下入炸好肉片翻炒，使每片均匀挂汁，出锅前加香菜点缀。‌


### 处理方法细节#
‌肉片处理‌：切片宜薄且大，腌制时间充足以确保入味。‌

‌挂糊均匀‌：淀粉糊浓度以抓起肉片糊不流淌为准，避免过厚影响口感。‌‌

‌油炸控制‌：两次油炸确保外焦里嫩，初炸熟透，复炸增酥脆；油温不足易软，过高易焦。‌‌
‌‌  
‌汁料平衡‌：糖醋比例可根据口味调整，传统哈尔滨派为酸甜口，辽宁派常加番茄酱。‌

‌
‌快速翻炒‌：肉片挂汁后迅速出锅，避免久炒使酥皮变软。‌‌
