# <center><span style="font-family: 黑体; font-size: 38px; ">**麻婆豆腐**</span>‌  #

---
<span style="font-family: 华文行楷; font-size: 24px; color: black;">菜名： 麻婆豆腐</span>

<span style="font-family: 华文行楷; font-size: 24px; color: black;">菜系： 川菜</span>

<span style="font-family: 华文行楷; font-size: 24px; color: black;">菜式： 热菜</span>

---

麻婆豆腐是一道享誉世界的经典川菜，以其独特的麻辣风味和丰富的文化内涵著称。
‌菜品历史与文化

‌
麻婆豆腐始创于1862年成都万福桥边的“陈兴盛饭铺”，由店主陈刘氏（因面有麻点，人称“陈麻婆”）创制。‌
 其烹饪技艺于2011年入选四川省非物质文化遗产代表性项目，并于2018年被评为“中国菜”四川十大经典名菜及世界川菜大会首批经典菜肴。‌
 该菜通过黄埔军校学员及海外厨师（如日本厨师陈建民）传播至全球，形成了多种本土化版本。‌

## <span style="font-family: 黑体; font-size: 20px; ">**核心风味特征**</span> ##

‌
传统麻婆豆腐以“麻、辣、烫、香、嫩、酥、鲜、活”八字著称。‌


‌麻‌：源于优质花椒（如汉源贡椒）研磨的花椒末。‌

‌辣‌：主要来自郫县豆瓣酱及辣椒的复合运用。‌

‌烫‌：豆腐保温性好，使菜品保持出锅温度。‌

‌香、酥、嫩、鲜、活‌：分别指复合香气、酥化肉末、嫩滑豆腐、新鲜原料以及蒜苗直立碗中的鲜活视觉效果。
‌

<span style="font-family: 黑体; font-size: 25px; ">经典家庭制作方法</span>

以下为一份基于传统工艺的家庭烹饪步骤。

‌<span style="font-family: 黑体; font-size: 25px; ">所需食材</span>‌

主料：豆腐1块（约300克）、牛肉末或猪肉末50克
关键调料：郫县豆瓣酱适量、豆豉适量、花椒粉（建议自制）、姜蒜末适量
辅料：料酒、生抽、淀粉、盐、糖、小葱、食用油、高汤或清水
‌制作步骤‌：

‌预处理‌：豆腐切成约1.5厘米见方的块，放入加有少许盐的开水中浸泡15分钟，以去除豆腥味并增加韧性。‌
干花椒用小火焙香后碾碎成花椒面备用。‌

‌炒制底料‌：锅中放油烧热，下姜蒜末煸香，加入切碎的豆瓣酱和豆豉，继续小火炒出红油和香味。‌

‌煸炒肉末‌：放入牛肉末或猪肉末，翻炒至变色、酥散，烹入料酒和少许生抽增香提色。‌

‌烧制豆腐‌：加入适量高汤或清水煮沸，将沥水的豆腐块轻轻滑入锅中。用勺背轻推，使豆腐均匀受热，中火烧约3-5分钟。‌

‌调味勾芡‌：根据口味调入少许糖和盐（注意豆瓣酱咸度），淋入水淀粉勾芡，使汤汁浓稠包裹豆腐。‌

‌成菜装盘‌：将豆腐盛入碗中，表面均匀撒上自制花椒面和新鲜小葱末。可选步骤：烧热少许食用油淋在花椒面和葱末上激发出香气。‌

‌<span style="font-family: 黑体; font-size: 25px; ">关键要点与演变</span> 
‌

‌肉末选择‌：传统版本使用牛肉末，后为适应更广食客，猪肉末也成为常见选择。‌

‌豆瓣酱‌：是形成“辣”与“香”风味的基础，建议使用正宗郫县豆瓣。‌

‌麻婆豆腐与麻辣豆腐区别‌：核心在于麻婆豆腐必须包含肉末。‌

‌国际演变‌：海外版本常在辣度、配料（如加入甜味）上进行本土化调整，但其麻辣烫香的核心辨识度得以保留
