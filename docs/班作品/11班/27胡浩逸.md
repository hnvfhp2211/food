# 回锅肉 川菜系菜品    

川菜回锅肉是川菜系经典菜品，属四川地方传统名菜，以精选肥瘦相连的五花肉为主料，搭配蒜苗等食材烹制而成，亦可添加青椒作为经典辅料。其制作需经煮、炒多道工序，通过旺火爆炒使肉片卷曲呈灯盏窝状，再佐以调味，蒜苗清香与可选辅料共同增添风味层次 [1-3]。
菜品兼具营养与开胃功效，猪肉富含优质蛋白质及血红素铁。成品肥而不腻的特质使其成为四川家常菜代表，广受食客喜爱。
中文名川菜回锅肉 
分    类川菜   
口    味麻辣香绝   
主要食材猪肉，蒜苗   
制作难度一般    


## 做法 ##
食材：猪肉、蒜苗、色拉油、姜、海椒、豆瓣酱、甜酱、味精。    
步骤：    
1、爆肉的一个要领，要置锅，就是把锅烧热，越热越好，主要是为了不让肉在爆的过程中贴锅；
2、爆肉。在爆之前可以加点色拉油进去，我今天买的肉有点偏瘦，没得好多油的。加点油更容易爆出油来；
3、肉爆好后，起锅，锅内留油。改小火煸炒泡姜、泡海椒、豆瓣。用小火的目的是为了不让佐料由于火大炒胡，让他的香味与味道都出来就行了；
4、佐料炒好后，肉回锅。回锅肉嘛，当然要多回几次锅了。这时再加甜酱，上色，当然加甜酱也是提味的一个关键，跟加糖是一个道理。怎么样，这时有颜色了；
5、加味精少许。这时肉基本已经炒好了，如果大家觉得差点味道还可以再加点盐；   
6、放入蒜苗，再加火小炒一下。注意，不要把蒜苗炒死了，炒生点更香！
