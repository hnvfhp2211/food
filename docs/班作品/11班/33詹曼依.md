‌
回锅肉是‌川菜的代表菜之一‌，被誉为“川菜之首”或“家常菜之首”，其历史渊源深厚，说法多样

‌起源时间可追溯至北宋‌，《东京梦华录》中记载的“爆肉”被认为与其有关；明代宋诩《竹屿房杂部》明确记载了“油爆猪”，已具备回锅肉的基本特征‌
2
3
。
‌清代咸丰年间（约1850年代）‌，郫县豆瓣酱创制并普及，成为回锅肉风味定型的关键调料‌
2
3
。
‌菜名首次见于文献‌是在1909年出版的《成都通览》中，被列为“家常便菜”‌
3
7
。
‌主流起源说法‌包括：
民间“打牙祭”习俗：每月初一、十五改善伙食，将祭祖后的熟肉切片回锅炒食‌
3
5
。
清末成都翰林凌元勋改良：将原“煮后炒”改为“蒸后炒”，锁住肉汁、提升香味‌
2
5
。
‌2018年入选“中国菜”四川十大经典名菜‌，2021年获评“天府名菜”‌

‌二、<span style="font-family: 华文琥珀; font-size: 20px; color: red;">做法</span>
‌
‌核心特点‌
‌主料‌：猪后臀“二刀肉”（肥四瘦六）或带皮五花肉
‌关键工艺‌：先煮后炒，‌二次烹调‌，肉片煸至卷曲呈“灯盏窝”状
‌灵魂调料‌：郫县豆瓣酱（需剁细）、豆豉（部分流派用）、甜面酱
‌经典配菜‌：青蒜（蒜苗），蒜白与蒜叶分次下锅
‌标准步骤‌
‌煮肉‌：带皮肉冷水下锅，加姜、葱、花椒、料酒，煮至八成熟（筷子可穿透但无血水），捞出晾凉‌
2
3
。
‌切片‌：趁外冷内热时切薄片（约0.3cm厚），可急冻2–3分钟更易操作‌
2
3
。
‌煸炒‌：热锅加少量油（或用熟猪油），中火煸炒肉片至吐油、边缘卷曲成“灯盏窝”‌
2
4
。
‌炒酱‌：将肉拨至锅边，下剁细的郫县豆瓣酱小火炒出红油，加豆豉、甜面酱、少许酱油或糖调味‌
2
4
。
‌加配料‌：大火先下蒜白段翻炒，再下蒜叶快炒断生，淋料酒提香，迅速出锅‌
2
‌关键技巧‌
‌选肉‌：必须当天宰杀的新鲜肉，肥瘦比例适中。
4
。
‌火候‌：煸肉用中火，炒酱用小火，最后快炒用大火。
‌调料‌：郫县豆瓣必须正宗，酱油需浓稠挂瓶壁‌
2
3







‌三、主要流派与变种 #‌
‌川西传统型‌（成都/乐山）：以蒜苗为主，咸鲜微辣，家常经典‌

。
‌川东旱蒸型‌：先蒸后炒，肉质更鲜香浓郁‌

。
‌现代变体‌：可搭配青椒、锅盔、苕皮、盐菜等，甚至有低脂健康版‌
