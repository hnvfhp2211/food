回锅肉的做法核心是“先煮后炒、二次回锅”‌，关键在于肉片煸炒至卷曲成“灯盏窝”状，搭配郫县豆瓣酱和蒜苗，香辣下饭，肥而不腻。

食材准备（1-2人份）‌
主料‌：五花肉 200-400g（优选肥瘦相间的二刀肉）
辅料‌：蒜苗 1把、青红椒 适量、洋葱 少许（可选）
调料‌：郫县豆瓣酱 10-15g、豆豉 少许、生姜 几片、大蒜 几瓣、料酒、生抽、白糖、盐、食用油
详细步骤‌
煮肉定型‌
五花肉冷水下锅，加入姜片、葱段、料酒和几粒花椒，大火煮15分钟左右，至筷子能轻松插入瘦肉部分即可。捞出后放凉，避免过熟变烂 。

切片处理‌
将晾凉的五花肉切成 ‌2-3mm厚的薄片‌，太厚油腻，太薄易焦。可提前将肉冷冻半小时更易切 。

煸炒出油‌
热锅不放油，直接下肉片中小火煸炒，炒至肉片卷曲、边缘微焦、肥肉透明出油，盛出备用 。

爆香调料‌
锅中留底油，下姜蒜片、豆豉和 ‌10ml郫县豆瓣酱‌ 小火炒出红油，注意火候别炒糊 。

回锅翻炒‌
倒入煸好的肉片，加少许料酒、生抽、一点点白糖提鲜，翻炒均匀。再加入青红椒、洋葱等配菜略炒断生 。

加入蒜苗‌
最后放入蒜苗段，快速翻炒10秒出锅。蒜苗不宜久炒，保持清香脆嫩口感 。

关键技巧与注意事项‌
选肉讲究‌：用带皮五花肉或猪后臀肉（二刀肉），层次分明更香 。
火候控制‌：煸肉用中火，炒酱用小火，避免焦苦。
调味平衡‌：豆瓣酱本身咸，建议最后尝味再决定是否加盐 。
正宗风味‌：可加少许甜面酱提升复合香味，是川味灵魂之一 。
