# 麻辣小龙虾 #

 ‌主料准备‌
小龙虾‌：2-3斤（建议选活虾，颜色鲜亮、腹部干净）

辅料‌：
葱段、姜片、蒜瓣 各适量
干辣椒 10-15个（根据辣度调整）
花椒 1大把（“大红袍”花椒更香麻）
八角 2-3颗、香叶 3片、桂皮 1小段、草果 1颗（拍碎）
郫县豆瓣酱 1大勺（红油版更佳）
生抽、老抽、蚝油、白糖、盐、鸡精 适量
啤酒 1罐（330ml）‌——去腥增香关键
可选：十三香粉、香辣小龙虾调料包（如秦妈、名扬等品牌）

‌第一步：处理小龙虾（关键！）‌
刷洗‌：用旧牙刷将小龙虾全身，尤其是头部与身体连接处刷洗干净。
去虾线‌：用手捏住尾部中间的尾鳍，轻轻一拉即可抽出黑色虾线。
去头（可选）‌：若追求入味，可剪掉头部前1/3（内含虾黄，也可保留）。
焯水（可选）‌：冷水下锅，加姜片、料酒，煮至变色捞出，去腥更干净 。

 ‌第二步：炸/煸小龙虾（锁色增香）‌
锅中倒油（约半碗），烧至五成热（150℃左右），放入小龙虾炸约1-2分钟至外壳变红，捞出控油 。
（也可省略油炸，直接炒制，但炸后更香且不易碎）

 ‌第三步：炒香料 & 炒虾‌
锅中留底油，小火炒香 ‌花椒、干辣椒、葱姜蒜‌。
加入 ‌郫县豆瓣酱‌ 炒出红油，再放入 ‌八角、香叶、桂皮、草果‌ 等香料炒香。
倒入小龙虾，大火翻炒均匀，让虾裹上香料味。

 ‌第四步：调味 & 焖煮（入味关键）‌
倒入 ‌1罐啤酒‌，没过小龙虾2/3即可（替代水，去腥提鲜）。
加入：
生抽 2勺、老抽 半勺（上色）
蚝油 1勺、白糖 1勺（提鲜）
盐、鸡精 适量
可选：十三香粉或整包小龙虾调料
大火烧开后，转中小火 ‌焖煮10-15分钟‌，让虾充分入味。
最后大火收汁，撒上 ‌香菜、白芝麻‌ 即可出锅 。
