#<center>跷脚牛肉</center>
![](https://gips1.baidu.com/it/u=3692758626,3266400069&fm=3074&app=3074&f=JPEG)
跷脚牛肉是四川省乐山市苏稽镇的传统名菜，起源于清末民初，因食客就餐时常跷脚而得名，以牛杂和中草药汤底为特色，兼具咸鲜麻辣风味与药膳功能。

##  <span style="font-family: 楷体; font-size: 24px; color: black;">起源与历史</span>
跷脚牛肉的起源存在两种主流说法，均指向乐山市苏稽镇。‌
‌

‌周天顺创制说‌：清朝光绪年间（约1875-1908年），苏稽镇周村屠夫周天顺将宰牛后废弃的牛杂收集起来，配以香料和中草药熬制成汤锅，在码头、河滩摆摊售卖，因价格低廉、味道鲜美，深受船工、脚夫等底层劳动者欢迎。‌
‌‌‌‌
‌罗老中医创制说‌：20世纪30年代初，乐山苏稽镇一位擅长中草药的罗姓老中医在河边悬锅烹药救济行人，偶然将丢弃的牛杂加入药汤中炖煮，发现汤味鲜香且具防病功效，食客络绎不绝。‌
‌‌
###  <span style="font-family: 楷体; font-size: 24px; color: black;">名字由来</span>

最初售卖跷脚牛肉的摊点条件简陋，仅有一张长条桌，无凳子供食客落座。食客们或站立、或蹲坐，常将一只脚跷在桌下的横梁上休息，久而久之便形象地称其为“跷脚牛肉”。‌
‌‌‌
####  <span style="font-family: 楷体; font-size: 24px; color: black;">菜品特点</span>
‌主要原料‌：以牛杂（如牛肚、牛肠、牛心等）为主料。‌

‌汤底工艺‌：配以多种中草药熬制，汤味咸鲜香浓，兼具散寒止咳、滋补养生的药膳功能。‌
‌
‌风味发展‌：从最初的汤锅摊逐渐演变为包含鲜烧、清烧、全牛席等系列的乐山特色美食，并于2008年被认定为乐山市非物质文化遗产。‌‌‌‌
