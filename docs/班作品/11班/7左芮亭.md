# <center>土豆丝</center>

土豆（马铃薯）起源于南美洲，约8000-10000年前被驯化，马铃薯传入我国的时间是在明朝万历年间，即1573年到1619年之间 	。炒菜技艺在南北朝时期就已经出现，并在明清时期大为流行 。炒土豆丝根植于鲁菜体系，成型于北方家常烹饪传统，以“脆、爽、淡、鲜”为特点，讲究“快火短炒、本味见功”。清代《调鼎集》已有“薯蓣切丝速炒”的记载；民国时期山东济南、潍坊一带的饭铺已将土豆丝列为“下酒三小碟”之一 。土豆丝的普及，既拜传统所赐，又与新中国的工业化息息相关。随着新中国煤矿的开发，城乡居民生火做饭的燃料问题逐步得到解决，猛火快炒不再只是饭馆的专利。同时，钢铁产量的提高使得轻薄、导热快的铁制炒锅进入寻常百姓家，加之土豆价格便宜，炒土豆丝遂成为最常见的一道家常菜
## 做法 #
选材：建议选用淀粉含量适中的土豆，如黄心土豆，避免使用发芽或表皮变绿的土豆。 
1.土豆去皮洗净，切成粗细均匀的粗丝，放在清水中浸漂，炒前捞出。
浸泡可以去除表面淀粉，防止氧化变黑，使炒制时更爽脆。 
焯水（可选）：部分做法建议焯水以进一步确保脆爽，但非必需。 
炒制：热锅冷油，油热后下干辣椒段、葱、姜、蒜等爆香。 
2.锅放在旺火上，倒入菜油，烧至六成热时，下豆瓣辣酱炒香至油成红色，放入土豆丝快速翻炒，加精盐、葱段、酱油、味精、少量鲜汤炒匀，炒至土豆丝刚熟，起锅装盘而成。
全程保持大火快速翻炒，时间不宜过长。 
调味可根据口味加入盐、醋等，醋可在出锅前淋入以增加香气。
当土豆丝变透明、断生时即可出锅
### 特点 ###
家常土豆丝是一道以‘脆、爽、淡、鲜’为核心风味的菜品。 它制作快捷，是家庭餐桌上的常见家常菜。口感上以‘脆、爽’为核心追求，通过切丝均匀、浸泡去淀粉、猛火快炒等工艺实现，切丝后浸泡冰盐水或加醋的清水是保证口感爽脆、防止氧化变黑的关键步骤风味基础为咸鲜本味，讲究食材本味与火候的转化，并衍生出酸辣、醋溜、酱香等多种经典口味，其中酸辣口味因其开胃下饭的特点尤为受欢迎，被誉为‘国民下饭菜’，调味时醋常分次或沿热锅边淋入以激发香气。 色泽讲究油亮诱人，土豆丝本身呈淡黄色或金黄色，常配以青红椒丝、胡萝卜丝等点缀，成品色泽光亮，美观可口。 菜品定位根植于鲁菜体系与北方家常烹饪传统，体现了‘快火短炒、本味见功’的技艺，其制作快捷、食材易得，承载着‘把普通食材做好’的朴素美味哲学与亲切的烟火气。
