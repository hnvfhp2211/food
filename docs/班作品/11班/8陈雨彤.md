# 回锅肉 #
[](https://bkimg.cdn.bcebos.com/pic/b999a9014c086e061d9551c8a9516cf40ad163d95da4?x-bce-process=image/format,f_auto/watermark,image_d2F0ZXIvYmFpa2UyNzI,g_7,xp_5,yp_5,P_20/resize,m_lfit,limit_1,h_1080)四川省传统川菜，被誉为川菜之首和家常菜肴代表，古称“油爆锅”，起源存在祭祀起源说、油爆猪演变说及袍哥发明说等多种说法 。
## 食材 ![](https://bkimg.cdn.bcebos.com/pic/86d6277f9e2f070828384b83427daf99a9014d0854a4?x-bce-process=image/format,f_auto/watermark,image_d2F0ZXIvYmFpa2UyNzI,g_7,xp_5,yp_5,P_20/resize,m_lfit,limit_1,h_1080)[](https://bkimg.cdn.bcebos.com/pic/86d6277f9e2f070828384b83427daf99a9014d0854a4?x-bce-process=image/format,f_auto/watermark,image_d2F0ZXIvYmFpa2UyNzI,g_7,xp_5,yp_5,P_20/resize,m_lfit,limit_1,h_1080)以猪后臀肉（二刀肉）为主料，经煮、煎、炒工艺制成，核心技法为“二次烹调”，调味以郫县豆瓣酱与豆豉为主 [9] [15]。其源头可追溯至北宋，《东京梦华录》记载的“爆肉”被认为与之相关，清末因豆瓣工艺改良提升风味 [4-5] [16]。该菜品需将猪肉煮至八成熟切片，煸炒至边缘卷曲呈“灯盏窝”状，常用青椒、蒜苗作配料，衍生出旱蒸香辣回锅肉、老咸菜回锅肉等变种 [9] [17]。调料选用家家常见食材，不同家庭风味各异，形成独特饮食文化特征。![](https://bkimg.cdn.bcebos.com/pic/b999a9014c086e061d9551c8a9516cf40ad163d95da4?x-bce-process=image/format,f_auto/watermark,image_d2F0ZXIvYmFpa2UyNzI,g_7,xp_5,yp_5,P_20/resize,m_lfit,limit_1,h_1080)
