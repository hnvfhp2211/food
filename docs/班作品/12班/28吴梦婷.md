# 经典沙拉食谱推荐
‌经典鸡胸肉果蔬沙拉‌：适合减脂人群，将鸡胸肉腌制煮熟撕块，搭配生菜、苦菊、圣女果、黄瓜及水果丁，淋上油醋汁拌匀，饱腹感强且低卡 。‌‌

‌金枪鱼土豆泥沙拉‌：将土豆蒸熟压泥，混合水浸金枪鱼、黄瓜丁、玉米粒及坚果碎，用黑胡椒和盐调味，口感绵密丰富 。‌‌

‌清爽蔬菜/水果沙拉‌：
蔬菜：紫甘蓝、生菜、胡萝卜等洗净切丝，可加煮鸡蛋或虾仁，搭配千岛酱或油醋汁 。‌

‌牛油果沙拉‌：牛油果切块与奶酪、玉米粒、生菜混合，使用橄榄油、苹果醋、盐和黑胡椒调制的油醋汁拌匀，需淋柠檬汁防氧化 。‌
