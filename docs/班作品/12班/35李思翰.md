# 菜品特色 #

蒜蓉炒花甲的口感鲜嫩多汁，花甲经过焯水处理后肉质紧实，不会因为长时间烹饪而变得老。其风味以蒜香浓郁为核心，经过煸炒的蒜蓉与花甲的鲜味融合；蒜分两次放入，一次爆香取醇厚，一次出锅前放保留生蒜辛辣，使口感层次丰富 。红辣椒和葱花的点缀使菜肴在视觉上更加诱人 。

## 做法 ##
吐沙需彻底，可采用盐水浸泡并滴入香油，或加醋等方法 ；冲洗前可将两个花甲互相搓洗检查，听不见沙沙声即为合格 。
需大火快炒，花甲开口后立即出锅，以防肉质因长时间加热而收缩变硬 。
调味时切忌额外放盐，因花甲本身有咸味 ；可加入半勺糖以提鲜并中和咸度。蒜宜分两次放入：第一次爆香以取醇厚香味，第二次在出锅前放入生蒜末以保留辛辣口感，增加层次 ![](https://pic.rmb.bdstatic.com/bjh/bc16a501d9d/250828/0f6fe761e7aca86f9c60cbb166c9c2dc.jpeg)。
关于预处理，常见方法包括焯水，花甲开口后需迅速捞出 ；亦有观点认为第一步焯水是错误的 。此外，也可采用蒸制开壳的方法 。
### 食用建议 ###
播报

蒜蓉炒花甲通常作为一道下饭菜，搭配米饭食用。需要注意的是，花甲属于海鲜，食用前需确保其新鲜，避免食物中毒。如果不喜欢太重的蒜味，可以在炒制时适当减少蒜蓉用量，或者在出锅前再加入一些生蒜蓉，以保留蒜的清香
