
# <center><span style="font-family: 华文琥珀; font-size: 20px; color: red;">钵钵鸡的来源</span>

钵钵鸡是起源于四川地区的传统小吃，属川菜系，以陶钵盛放去骨鸡片并配麻辣调料为特色，具有皮脆肉嫩的特点。其名称源于盛装容器“钵钵”，食用时将竹签串制的食材浸入藤椒油汤或红油汤料中自取，常与奶汤面搭配形成独特风味 该小吃由叶春兰于1980年在乐山犍为县首创，早期以“椒麻鸡”形式分块售卖，1997年改为竹签串制并更名为“串串鸡”，2014年正式定名“钵钵鸡” [![](https://qcloud.dpfile.com/pc/lUbQg_qyZjvXoTNcM7og5Uz6Jk2f7p6pU8DugHj4nasfc-dMJkD_5syX1fNWrwV8.jpg)3]。
###<span style="font-family: 华文琥珀; font-size: 20px; color: red;"><center>钵钵鸡的特点</span>
钵钵鸡的特点 ###
该小吃以冷泡入味为特点，需将食材在红油或藤椒油汤汁中浸泡20-30分钟，核心调料包括商用型红油、藤椒油和复合汤汁 [5]。1990年获成都市个体名小吃优质奖，1991年被成都市人民政府命名为优质名小吃，2022年2月入选国家《地标美食名录》 [1-2]。现代销售模式已从瓦罐冷串串发展为超市称重销售，2023年海底捞曾推出夜市钵钵鸡摊位 [5]。
####<span style="font-family: 华文琥珀; font-size: 20px; color: red;"><center>钵钵鸡的食材</span> ####
‌一、主料与配料准备‌
‌主料‌：以鸡肉为主，常用仔公鸡、三黄鸡或跑山鸡；也可用鸡腿、鸡翅、鸡胗等部位。
‌荤菜搭配‌：鸡爪、鸡尖、鸭血、鹌鹑蛋、火腿肠、丸子、豆皮等。
‌素菜搭配‌：藕片、土豆、西兰花、木耳、海带结、腐竹、金针菇
‌关键调料‌：红油辣椒、藤椒油、花椒粉、熟芝麻、生抽、蚝油、白糖、盐、姜葱蒜、香油等‌
