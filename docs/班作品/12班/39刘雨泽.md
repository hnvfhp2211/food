# 鱼香肉丝的由来 # 

                                       鱼香味儿肉丝，通常指鱼香肉丝，是一道经典的川菜，属于鱼香味的代表菜肴。其名称源于模仿四川民间烹鱼所用的调料（如泡椒、姜、蒜、糖、醋等）调和出的独特复合味型，成菜并无鱼肉。，常配以木耳丝、竹笋丝等辅料。核心在于使用泡!!!
/##鱼香肉丝的做法  ##
以猪里脊肉为主料，配以泡辣椒、姜、蒜等调料炒制而成。该菜品起源于民国时期，相传由民国时期川菜厨师在泡椒肉丝基础上创新而成，1947年其制作方法首次正式记载于重庆《新华日报》 。其味型通过不含鱼的调料复现鱼香风味，此法源自重庆民间烹鱼技法，呈现咸、酸、辣、甜、鲜五味调和的特征，成菜色泽红润且葱姜蒜香浓郁。
### 鱼香肉丝的资料 ###
鱼香肉丝（Yuxiang shredded pork） ，是中国四川特色名菜，以猪里脊肉为主料，配以泡辣椒、姜、蒜等调料炒制而成。该菜品起源于民国时期，相传由民国时期川菜厨师在泡椒肉丝基础上创新而成，1947年其制作方法首次正式记载于重庆《新华日报》]。其味型通过不含鱼的调料复现鱼香风味，此法源自重庆民间烹鱼技法，呈现咸、酸、辣、甜、鲜五味调和的特征，成菜色泽红润且葱姜蒜香浓郁。制作时选用猪肥瘦肉按3:7比例切二粗丝，配泡红辣椒及木耳、竹笋等辅料，采用急火短炒、一锅成菜的技法。肉丝需经码味、码芡后炒至散籽发白，再依次加入调料翻炒收汁。2018年入选"中国菜"四川十大经典名菜名单 ，2021年入选省级"天府名菜"第一名 ![](https://q5.itc.cn/images01/20240926/97e6d4905700474ebfe80cfc9a5451f9.jpeg)
