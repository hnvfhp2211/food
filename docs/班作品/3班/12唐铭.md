# 红烧肉
 
![](https://p3.ssl.qhimgs1.com/t011f7e32ffe92d5853.jpg)
红烧肉是一道经典的中国家常菜，以下是综合多个来源整理的红烧肉做法：

选材：选择肥瘦相间、层次分明的五花肉，肥瘦比例在3:7到4:6之间较为合适12。
处理五花肉：将五花肉切成2-3厘米见方的块，冷水下锅，加入姜片、葱段和料酒，煮沸后撇去浮沫，捞出洗净沥干备用123。
炒糖色：锅中放入冰糖，小火慢慢炒化成深琥珀色的糖色，迅速倒入沥干水分的五花肉翻炒均匀，让每一块肉都裹上糖色345。
加入调料：放入姜片、葱段、八角、桂皮、香叶等香料，翻炒出香味后，加入生抽、老抽继续翻炒均匀12。
炖煮：加入热水，水量要没过五花肉，大火烧开后转小火慢炖40分钟到1个小时，或根据个人喜好调整时间。炖煮过程中不要频繁打开锅盖，以免热量散失，影响炖煮效果124。
收汁：炖煮至肉质软烂后，根据口味加盐调味，开中火收汁，收汁时要时不时翻炒一下，让每块肉都裹上浓稠的汤汁，直到汤汁变得浓稠，能挂在肉表面即可关火2。

# 红烧肉的由来 #
 苏东坡与东坡肉：红烧肉的起源与北宋大文豪苏东坡密切相关。苏东坡曾任杭州知州，他组织民工疏浚西湖，并将猪肉切成方块，以姜、葱、糖、酒等调料炖煮，分给民工们食用。这道菜因肉色红亮，口感醇厚，深受喜爱，后人称之为“东坡肉”，是红烧肉的雏形之一。此外，苏东坡在被贬黄州时，也因烹饪猪肉并写诗推广，使得红烧肉的做法在当地百姓间传开123。

寒食节起源说：春秋时期，晋国大夫介子推曾随晋文公重耳历经患难。重耳归国为君后，介子推却隐居山林。文公欲逼其出山，放火烧山，介子推抱树不出，最终被火焚身。晋文公深感懊悔，下令全国在介子推忌日禁火，只吃冷食，这便是寒食节的由来。寒食节期间，人们无法生火做饭，只能吃冷食，而有一种说法是，后来人们在祭祀介子推时，会用猪肉等祭品，祭祀完毕后，将这些猪肉进行烹制，逐渐演变成了红烧肉，以纪念介子推的忠诚与高洁1。

历史文献记载：北魏贾思勰的《齐民要术》中记载了红烧肉的具体做法，这是目前传世文献中的最早记录。到了明朝，随着烹饪技术的发展，红烧肉开始在民间流传开来。清朝时期，红烧肉已经成为了宫廷菜的代表之一，乾隆皇帝非常喜欢吃红烧肉456。

综上所述，红烧肉的来历有多种说法，其中以苏东坡与东坡肉的故事最为人所熟知。红烧肉不仅是一道美味佳肴，更是一种文化传承，体现了中国烹饪技艺的精湛和人们对美好生活的追求。
