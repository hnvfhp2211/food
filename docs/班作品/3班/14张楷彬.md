# 红烧肉 #
红烧肉‌是一道经典的中式家常菜，以五花肉为主料，经过焯水、炒糖色、慢炖等步骤，成品色泽红亮、肥而不腻、入口即化，是许多家庭餐桌上的“下饭神器” 。

这道菜在中国各地有多种流派，最具代表性的包括：

上海本帮红烧肉‌：浓油赤酱，偏甜，体现“甜出头，咸收口”的风味；
湖南毛氏红烧肉‌：不加酱油，靠炒糖色上色，色泽枣红，香而不腻；
东坡肉‌：源自北宋苏东坡，讲究“慢着火，少着水，火候足时它自美”，口感软糯醇厚 。![](https://qcloud.dpfile.com/pc/gozx53dxUuOjBJKf5pMs7iCCehyi_KZPZd7hWKi0Os-K3R5Ac8sPJZic56Lvm_q0.jpg)
## 制作要点（基于国宴级技巧总结）： ##!
1.选材讲究‌：选用肥瘦相间的三层五花肉，带皮更佳；


2.预处理去腥‌：冷水下锅焯水，加姜片、料酒，撇去浮沫后温水洗净；

3.煸炒出油‌：干锅煸炒五花肉，逼出多余油脂，是“肥而不腻”的关键；

4.炒糖色增亮‌：小火炒冰糖至枣红色，迅速倒入肉块翻炒上色，比单纯用老抽更自然红亮；

5.慢火炖煮‌：加开水没过肉块，放入葱姜、八角、桂皮等香料，小火炖1小时以上；

6.调味时机‌：盐和生抽建议在炖煮40分钟后加入，避免肉质变柴；

7.收汁提味‌：最后大火收汁，不停翻炒，让酱汁浓稠均匀包裹肉块 。
