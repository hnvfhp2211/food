# <center>麻婆豆腐</center> #
#![](https://img0.baidu.com/it/u=2810851888,2387171068&fm=253&fmt=auto&app=138&f=JPEG?w=1310&h=800)  #
麻婆豆腐乃四川地区名菜之一，家常麻婆豆腐‌是一道经典川菜，麻辣鲜香，豆腐嫩滑，非常适合搭配米饭。下面分享一个简单易做的家常做法：

‌食材准备‌（2-3人份）：
‌主料‌：嫩豆腐（南豆腐）1块（约400g）、猪肉末100g（可选牛肉末）
‌辅料‌：蒜末2瓣、姜末1小勺、葱花适量
‌调料‌：
郫县豆瓣酱1大勺
辣椒粉1小勺（根据口味调整）
花椒粉1/2小勺（或花椒油）
生抽1小勺、老抽少许（调色用）
糖1/2小勺、盐少许（豆瓣酱咸，注意减盐）
水淀粉（1勺淀粉+3勺水调匀）
食用油适量
‌步骤‌：
‌处理豆腐‌：
豆腐切2cm见方的小块，放入加盐的沸水中焯1分钟（去豆腥且不易碎），捞出沥干备用。

‌炒肉末‌：
热锅冷油，中小火煸炒肉末至变色，加入姜蒜末炒香。

‌炒香调料‌：
转小火，加郫县豆瓣酱炒出红油，放辣椒粉、花椒粉炒匀（火别大，避免焦糊）。

‌炖豆腐‌：
➤ 倒入豆腐，轻轻推动混合（别大力会碎）。
➤ 加生抽、老抽、糖和少许热水（没过豆腐一半），中小火煮3分钟入味。

‌勾芡‌：
淋入水淀粉，轻推至汤汁浓稠，尝味补盐（豆瓣酱已咸，慎加）。

‌出锅‌：
装盘撒花椒粉、葱花，完成！

‌关键技巧‌：
‌豆腐不碎‌：焯水时加盐，烹饪时少翻动，用锅铲轻推。
‌麻辣调整‌：怕辣可减辣椒粉，喜麻可多放花椒粉或最后淋花椒油。
‌升级版‌：可加豆豉或蒜苗提香，用牛肉末更传统。
这道菜麻辣过瘾，配米饭绝佳。
