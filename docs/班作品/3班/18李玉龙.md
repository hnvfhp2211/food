‌西湖醋鱼
# ### **** ### #的正宗做法讲究选料、刀工、火候与糖醋汁的黄金比例，核心是“饿养去腥、七刀半改刀、沸水汆鱼、浇汁滋啦”四步精髓‌。这道菜色泽红亮，鱼肉鲜嫩，酸甜适口，带有独特的“蟹味”，是杭帮菜的代表之作。

一、历史渊源与正宗定义
西湖醋鱼起源于南宋，前身为“宋嫂鱼羹”，后在清代发展为“醋搂鱼”，最终定型于民国时期的杭州楼外楼 。如今公认的正宗做法，以‌草鱼‌为主料，经“饿养”净化泥腥味，配合精细刀工与恰到好处的火候，还原“肉似蟹肉”的独特口感 。

虽然现代也有用笋壳鱼、鳜鱼等改良版本，但传统杭帮菜馆仍以西湖草鱼为正宗之选 。

二、正宗做法四步核心流程
1. ‌选鱼与处理：饿养去腥‌
选用‌1.2–1.5斤的鲜活草鱼‌，过大则肉老，过小则刺多 。
‌提前在清水中饿养1–2天‌，让鱼吐尽泥沙，肉质更紧实 。
宰杀后彻底清洗，去除血水和黑膜。
2. ‌刀工讲究：七刀半改刀法‌
将鱼从尾部入刀，沿脊背剖成雌雄两片（雄片带脊骨，雌片为净肉），‌背部相连不断‌。
在雄片肉厚处斜切五刀，雌片切三刀，形成“七刀半”刀法，使鱼肉受热后自然卷曲成牡丹状 。
刀口深度约2cm，间距均匀，是保持形态美观的关键 。
3. ‌火候控制：沸水汆熟‌
锅中加水1000ml，放入姜片、葱段、料酒，烧至沸腾。
先放鱼头，再将鱼身并列放入，鱼皮朝上，保持文火微沸，‌汆烫3–5分钟‌。
用筷子轻扎鱼肉，能轻松插入即熟，捞出控水装盘 。
4. ‌糖醋汁：黄金比例，淋汁“滋啦”‌
‌糖醋汁比例：糖:醋 = 1:1‌，常用白糖60克、米醋50克、酱油75克、料酒25克、姜末1.5克、湿淀粉50克 。
做法：锅中留250ml原汤，加入调料烧开，勾薄芡至浓稠如镜面。
‌关键一步：将热汁淋在鱼身上时，必须听到“滋啦”一声‌，这是风味激发的标志 。
三、风味特点与文化意义
‌色泽红亮，酸甜适中，鱼肉嫩滑，带有“蟹味”‌，是其最大特色 。
被誉为“叔嫂传珍”，承载着亲情与坚韧的民间传说 。
2018年被评为‌浙江十大经典名菜之一‌，是杭州饮食文化的象征 。
