# 豆腐脑 #

一、基础材料准备‌
主料‌：干黄豆 150克（提前浸泡8-10小时，至无硬芯）
水‌：1500毫升（黄豆与水比例约1:10）
凝固剂‌：葡萄糖酸内酯 3克（即“豆腐王”，食品添加剂店或电商平台有售）
二、详细制作步骤‌
打磨豆浆‌

将泡发黄豆洗净，分次加入破壁机/豆浆机，加水打成生豆浆。
用纱布或滤网过滤豆渣，确保豆浆细腻 。
煮豆浆‌

将过滤后的豆浆倒入锅中，大火煮开后转小火继续煮3-5分钟，边煮边搅拌防糊底。
撇去浮沫，关火后静置降温至 ‌85℃–90℃‌（无温度计可等待3–5分钟） 。
点浆凝固‌

提前将内酯用少量凉开水化开，倒入干净容器中。
将热豆浆从高处冲入内酯液中，‌一次性倒完，不要搅拌‌。
立即盖上盖子，静置 ‌15–20分钟‌，让蛋白自然凝固 。
成品状态‌

成功的豆腐脑应呈果冻状，颤而不散，挖起有弹性 。
三、调味推荐（咸甜皆宜）‌
甜口‌：加白糖、蜂蜜或红豆汤。
咸口‌：搭配卤汁食用更地道，做法如下：
热油炒香肉末、葱姜蒜，加入泡发的黄花菜、木耳、香菇翻炒。
加水烧开，调入酱油、盐、胡椒粉，水淀粉勾薄芡，最后淋入蛋液搅成蛋花 。

豆腐脑的做法关键在于精准控制豆浆温度与凝固剂比例，让蛋白自然凝结成嫩滑豆花‌。结合多份实践验证的配方，以下是清晰、可复现的家庭操作指南：

一、核心材料配比（成功率高）‌
干黄豆‌：‌150克‌（提前浸泡8–10小时，至掐开无硬芯）
清水‌：‌1500毫升‌（豆水比约1:10，保证口感细腻）
葡萄糖酸内酯‌：‌3克‌（即“豆腐王”，电商平台或调料店可购）
✅ 小贴士：使用内酯比石膏更易掌控，成品无涩味，适合家庭操作。

二、分步操作流程‌
磨浆过滤‌

将泡好的黄豆加水放入破壁机，打成生豆浆。
用细纱布或滤袋过滤豆渣，‌滤得越净，豆腐脑越滑‌。
煮浆控温‌

豆浆倒入锅中，大火煮沸后转小火再煮3–5分钟，彻底去除豆腥并杀菌。
煮时不断搅拌防糊底，关火后撇去浮沫，‌静置降温至85℃–90℃‌（手感烫但不灼手）。
点浆成型‌

提前将3克内酯用‌20毫升凉白开‌化开，倒入干净容器。
从高处将热豆浆冲入内酯液中‌，利用冲力混合均匀，‌切勿搅拌‌。
盖上盖子，‌静置15–20分钟‌，期间避免晃动，待其自然凝固。
成品判断‌

成功的豆腐脑呈‌果冻状、颤而不散‌，用勺轻挖有弹性，豆香浓郁。
三、风味搭配建议‌
甜口派‌：加白糖、蜂蜜、桂花糖浆，或搭配水果丁、红豆沙。
咸口派‌（北方经典）：
热油爆香葱姜，炒肉末、香菇、木耳，加水烧开。
调入酱油、盐、胡椒粉，水淀粉勾薄芡，淋蛋液成蛋花，最后撒香菜。
