# 回锅肉 #
<span style="font-family: 楷体; font-size: 24px; color: black;">名称：回锅肉

<span style="font-family: 楷体; font-size: 24px; coblack;">菜系：川菜

<span style="font-family: 楷体; font-size: 24px; color: black;">菜式：热菜</span>


回锅肉是川菜的代表之作，被誉为“川菜之首”和“家常菜之王”，以其色泽红亮、肥而不腻、咸鲜微辣略带回甜的独特风味深受喜爱‌ 。这道菜的核心在于“二次烹调”——先将猪肉煮至八成熟，切片后再回锅爆炒，使肉片边缘微卷，形似“灯盏窝”，搭配郫县豆瓣酱与蒜苗，香气扑鼻，是四川人家家户户都能做的一道“过门香” 。

起源与文化
回锅肉的历史可追溯至北宋时期的“爆肉”或明代的“油爆猪”，但其现代风味的定型与清代咸丰年间‌郫县豆瓣酱‌的创制密不可分 。民间流传最广的说法是，它起源于四川的祭祀习俗“打牙祭”：祭祖后的熟肉冷却后难以下咽，人们便将其切片回锅炒制，加入蒜苗、豆瓣等调料，意外成就了一道美味佳肴 。因此，回锅肉不仅是一道菜，更承载着川人对家庭、节庆与乡土的记忆 。

主要食材与流派
主料‌：传统选用猪后臀肉（俗称“二刀肉”），肥瘦相间，带皮为佳 。
核心调料‌：郫县豆瓣酱、豆豉、甜面酱、蒜苗（或青椒） 。
地域流派‌：
连山回锅肉‌：源自四川广汉连山镇，以大片肉著称，2006年其制作技艺被列入德阳市非物质文化遗产 。
久蒸回锅肉‌：清末成都翰林凌元勋改良，先蒸后炒，肉质更鲜香 。
其他变种：干豇豆回锅肉、蕨菜回锅肉、红椒回锅肉等，体现川菜“一菜多味”的特点 。
制作关键
煮肉‌：冷水下锅，加姜、葱、花椒去腥，煮至七八成熟（断生即可），捞出晾凉切片 。
煸炒‌：旺火煸出肉中油脂，使肉片呈“灯盏窝”状，避免过油或焦糊 。
调味‌：先炒香郫县豆瓣酱出红油，再下肉片，依次加入甜面酱、豆豉、酱油，最后放入蒜苗快炒断生 。
2018年，回锅肉入选“中国菜”四川十大经典名菜；2021年获评“天府名菜”，成为四川饮食文化的标志性符号 。
