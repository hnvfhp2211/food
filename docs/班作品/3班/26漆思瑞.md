# 糖醋排骨 #

糖醋排骨（Sweet and Sour Spare Ribs）是以猪肋排或子排为主料，采用糖醋调味烹制的传统名菜，起源于浙江，属浙菜经典，亦见于淮扬菜等多地区菜系 [3-5] [9-10] [16] [18]。成菜色泽红亮油润，肉质鲜嫩，酸甜适口，兼具家常烹饪与宴客功能 [3] [9] [12] [15]。其历史渊源可追溯至南北朝时期的“酸豚”记载，宋代蔗糖普及后糖醋口味逐渐形成物质基础，明清时期定型为现代常见做法 [19]。

##主要工艺  ##

主要工艺包含焯水去腥、炒糖色或煎炸上色、糖醋汁焖煮及收汁等环节，常用调料有冰糖、香醋、料酒、生抽等，现代创新做法会使用番茄酱或按“12345”黄金比例（1酒2抽3醋4糖5水）调制料汁 [5-7] [11] [14-15] [24-25]。排骨需经焯水去除血水，通过干煸或油炸使表面金黄，加入糖醋汁焖煮至骨酥肉嫩，收汁后撒芝麻、葱花装饰 [7-8] [11] [13] [15]。不同地域存在做法差异，如上海版使用番茄沙司，四川版调整糖醋比例，现代还发展出话梅糖醋排骨等创新风味 [16-17] [22-23]## 主要工艺 ##

### 食用指南 ###

糖醋排骨虽然味美易消化，易吸收，但排骨本身油脂含量较高，食用尽量和其他蔬菜类辅食、谷物类主食一起进餐。不可单一大量贪食，以免脂肪积。
