# 鱼香肉丝 #
---
# 名称：鱼香肉丝 #
# 菜系：川菜 #
# 菜式：热菜 #

---

## 第一步：准备工作 - 磨刀不误砍柴工 ##

切切切： 把猪里脊肉先切成薄片，再切成细丝。记住一句话：“横切牛羊竖切猪”，切猪肉的时候，要顺着肉的纹理切，这样炒出来的肉丝才不会散。然后把青椒、胡萝卜、泡发好的木耳都切成和肉丝差不多粗细的丝。笋丝如果是袋装的，冲洗一下沥干水分就行。姜、蒜、泡椒统统切成末。

嫩肉技法： 把切好的肉丝放入碗中，加入1勺料酒、1勺生抽，用手充分抓匀，让肉丝把汁水完全“吃”进去。一直抓拌到能感觉到有点发粘了，再加入1勺淀粉，继续抓匀，直到每根肉丝的表面都能裹上一层薄薄的浆。最后淋上半勺食用油，再次抓匀。这一步叫作“封油”，能锁住水分，炒的时候也不容易粘成一团。处理好后，放在一边腌制至少15分钟。

调制灵魂酱汁： 拿一个小碗，按照上面“灵魂鱼香汁”的比例，把除了盐之外的所有调料统统放进去，然后用勺子搅拌均匀，特别是要把沉底的淀粉搅开，变成一碗顺滑的汁水。这一步千万不能省！ 如果等炒的时候再一样样往里加，手忙脚乱不说，火候也容易过头。

## 第二步：滑炒肉丝 - 要嫩，不要老 ##

锅烧热，倒比平时炒菜多一点的油。油温不用太高，看到锅底有微微的油纹时，就把腌好的肉丝倒进去。

用筷子快速地把肉丝划散，让它们均匀受热。看到肉丝刚刚全部变白，立刻就用漏勺捞出来，沥干油。注意！是“变白”就出锅，不是变金黄！ 这时候的肉丝大概有8成熟，内部还是嫩的，后续还要回锅，如果现在炒老了，这道菜就失败了。


## 第三步：炒香酱料 - 激发“鱼香”的灵魂 ##

锅里留底油，如果刚才的油多了就倒出去一些。开中小火，放入那一大勺郫县豆瓣酱和切好的泡椒末，慢慢炒。

一定要炒出红油，并且能闻到豆瓣和泡椒混合的那种独特的、醇厚的香气。这个过程大概需要1-2分钟，耐心点，这是味道层次的关键。

![](https://img1.baidu.com/it/u=925773698,1763578397&fm=253&app=138&f=JPEG?w=800&h=1005)
## 第四步：混合翻炒 - 上演终极融合 ##

香味出来后，把姜末、蒜末倒进去，翻炒几下爆出香味。

接着，把胡萝卜丝、青椒丝、木耳丝、笋丝这些比较难熟的蔬菜先倒进去，转大火，快速翻炒大约1分钟，让它们断生。

蔬菜炒得差不多了，就把之前滑炒好的肉丝倒回锅里，快速翻炒均匀，让肉丝和蔬菜都裹上酱料的颜色。

关键时刻！ 再次搅动一下我们提前调好的那碗“灵魂鱼香汁”（因为淀粉会沉淀，所以要搅拌一下），然后沿着锅边把香汁倒进锅内。这个时候，你会听到“刺啦”一声，伴随着一股浓郁的醋香和酱香升起，这就是所谓的“锅气”！

迅速颠锅或者用铲子快速翻炒，你会发现锅里的汤汁在淀粉的作用下，迅速变得浓稠，明亮地包裹在每一根肉丝和蔬菜上。


## 第五步：出锅装盘 - 准备消灭米饭！##

当汤汁完全包裹住所有食材，颜色红亮，香气扑鼻的时候，关火！撒上一点葱绿点缀，立刻出锅装盘！

看，一盘色泽红润、香气诱人、肉丝嫩滑、配料爽脆的鱼香肉丝就做好了！赶紧去盛一碗热气腾腾的白米饭吧！
