鱼香茄子是一道经典的川菜，以其“不见鱼却有鱼香”的独特风味而闻名，口感软糯入味，酸甜微辣，超级下饭‌ 。

这道菜属于“鱼香味型”，核心在于用泡椒、豆瓣酱、姜、蒜、糖、醋等调料调出复合味道，模拟出烧鱼时的香气。虽然名字里有“鱼”，但并不含鱼肉，而是源于四川民间用烹鱼余料炒茄子的智慧，早期曾被称为“余香茄子”，后演变为“鱼香茄子” 。

<span style="font-family: 微软雅黑; font-size: 18px; color: #165DFF;">做法</span>


鱼香茄子的家常做法要点
食材准备‌
主料：紫皮长茄子2-3根、猪肉末20-50g
辅料：姜蒜末、葱花、小米辣（可选）、青红椒丝
核心调料：郫县豆瓣酱、生抽、老抽、醋、白糖、盐、水淀粉

茄子预处理（关键步骤）‌

茄子切条后，用盐腌制10-15分钟，挤出水分，可有效减少吸油量 。
也可在茄条表面裹一层薄淀粉，炸制时形成保护层，更健康少油 。
调制鱼香汁（灵魂所在）‌
推荐“54321”黄金比例：

5勺清水 + 4勺醋 + 3勺生抽 + 2勺糖 + 1勺淀粉‌，搅拌均匀备用 。
可根据口味微调，喜欢更甜可加糖，偏酸则多加醋。
烹饪步骤‌

热锅凉油，中高油温下茄条炸至微黄定型，捞出控油（也可用煎代替炸）。
锅中留底油，小火炒香豆瓣酱和姜蒜末，炒出红油。
加入肉末炒散至变色。
倒入炸好的茄子，翻炒均匀。
淋入调好的鱼香汁，大火收汁，最后沿锅边烹入少许醋提香。
撒上葱花即可出锅。

<span style="font-family: 微软雅黑; font-size: 18px; color: #165DFF;">特点</span>


鱼香茄子的核心特色
独特的“鱼香味型”‌
这是川菜24种经典味型之一，讲究“咸、甜、酸、辣、鲜、香”六味融合，以郫县豆瓣酱和泡红椒为基底，辅以姜、蒜、葱“三件套”激发浓郁香气 。成菜后红亮油润，酱汁包裹茄条，达到“见油不见汁”的标准 。

口感层次丰富‌
茄子经炸或煎处理后外微焦里软糯，吸饱鱼香汁后入口即化，搭配肉末的香酥与葱蒜的辛香，每一口都充满满足感 。

家常可做，变化多样‌

经典做法多用油炸茄子锁形减油，现代健康版则可用蒸、煎或空气炸锅替代 。
素食者可省去肉末，加入杏鲍菇或木耳提升口感；控糖人群也可用代糖调整鱼香汁比例 。
创新吃法如“鱼香茄子煲”“凉拌鱼香茄子”也逐渐流行，适应不同季节与口味需求 。
文化意义深远‌
鱼香茄子体现了川菜“化平凡为神奇”的智慧，从一道节俭菜发展为全国乃至国际知名的中华美食代表之一，在TikTok、YouTube等平台广受海外食客喜爱 。它不仅是餐桌上的美味，更是中华饮食文化的生动载体。
