# 麻婆豆腐 #
麻婆豆腐（Mapo Doufu），川菜传统名菜，始创于1862年成都万福桥陈兴盛饭铺，由店主陈刘氏（因面有麻点称陈麻婆）创制，以麻、辣、烫、香、嫩、酥、鲜、活八字特点著称，2011年其制作技艺入选四川省非物质文化遗产代表性项目 [3] [7-9]。
该菜以豆腐与肉末为主料，配豆瓣酱、花椒等调料，经黄埔军校学员传播至全国及海外 [5]。日本厨师陈建民1958年创办四川饭店推出改良版麻婆豆腐，后通过《四川料理之旅》普及日本，美国、意大利等地亦有本土化版本 [6] [14-15]。
2018年入选“中国菜”四川十大经典名菜及世界川菜大会首批经典菜肴 [1] [4] [10]，2021年获评“天府名菜” [11]。日本东京等五座海外城市设立川菜推广中心，非遗传承人参与国际推介 [12-13] [16]。

 麻婆豆腐是川菜经典代表菜肴，其特色鲜明，以“麻、辣、烫、香、酥、嫩、鲜、活”八字精髓著称 [8]，亦有“七味一体”之说。追求诸味平衡，麻而不燥、辣而回甘、酥香衬嫩鲜 [4]。成菜色泽红亮，红白绿三色相间，白的是豆腐，红的是辣椒（或辣椒油），绿的是蒜苗 [6]。其独特风味依赖于郫县豆瓣酱、汉源花椒、豆豉等核心调料，体现了川菜麻辣味型的精髓 [4] [6-7]。作为国际名菜，麻婆豆腐衍生出多种创新形式 [10] [12]。2026年

1月，日本罗森推出了冰镇“可饮用麻婆豆腐”饮料，尝试以饮品形式还原其麻辣口感与嫩豆腐的滑嫩 [3] [11]。

关于麻婆豆腐的研究涵盖了其科学原理与文化传播等多个方面 [7] [13-14]。
科学解析揭示了麻婆豆腐风味的奥秘 [7]。例如，豆腐通过盐水浸泡，可以使大豆蛋白部分变性，从而在烹饪时更好地保持形状 [4]。其独特的“麻”与“辣”源于花椒中的羟基-α-山椒素与辣椒中的辣椒素，它们分别激活不同的神经受体，产生振动麻感与灼烧辣感，形成“麻辣共振” [7]。
麻婆豆腐的跨文化传播，尤其在日本的发展历程，是重要的研究课题。1952年，川厨陈建民东渡日本，后在东京开设四川饭店，将麻婆豆腐引入日本。为适应本土口味，他通过降低辣度、增加甜味、融入柴鱼高汤等方式进行改良，使其逐渐成为国民料理。1964年，陈建民在NHK《今日料理》节目中的演示，曾引发日本全国的豆腐抢购潮。媒体将麻婆豆腐在日本的社会与经济影响称为“麻婆豆腐经济学”，指出其已成为家庭餐桌的“情绪稳定器” [13]。
当代的麻婆豆腐研究也关注其创新形式与文化反哺。日本市场衍生出大量跨界产品，而成都的陈麻婆豆腐总店则推出了“东京限定版”，在保留传统技法的基础上，融入本味醂与木鱼花等日式元素，形成了“逆向本土化”现象

麻婆豆腐已成为国际名菜。 [12]1952年，川厨陈建民将麻婆豆腐引入日本。 [10]随后流行范围扩展至美国、加拿大、英国、法国、越南、新加坡、马来西亚、澳大利亚等国。2018年9月10日，麻婆豆腐被“中国菜”项目评为四川十大经典名菜。 [12]
在日本，麻婆豆腐经历了显著的本土化过程。1964年，陈建民在NHK节目《今日料理》中示范制作方法，引发了全国性的豆腐抢购潮。通过降低辣度、增加甜味、融入柴鱼高汤等改良，麻婆豆腐完成了从异邦料理到国民料理的蜕变。日本外务省调查显示，87%的国民每月至少食用两次麻婆豆腐。 [13]衍生产品方面，1971年，丸美屋食品工业率先开发了供家庭食用的麻婆豆腐酱料包。 [10]2026年1月，日本罗森便利店推出了冰镇“可饮用麻婆豆腐”饮料，成为其“飲む○○シリーズ”的第31款产品。 [3]此外，漫画《中华一番！》及富士电视台节目《料理铁人》等媒体作品也促进了麻婆豆腐的文化渗透与流行
