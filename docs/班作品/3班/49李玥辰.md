# **锅包肉** #
1. # 锅包肉是起源于清朝光绪年间哈尔滨道台府的一道经典东北菜，由厨师郑兴文为迎合俄罗斯宾客口味，将原本咸鲜的“焦烧肉条”改良为酸甜口的“锅爆肉”，后因发音演变成为“锅包肉”‌ 。这道菜以猪里脊肉为主料，经切片腌制、裹土豆淀粉糊、两次油炸后，再与糖醋汁快速翻炒而成，成品色泽金黄、外酥里嫩、酸甜可口，是东北菜系中极具代表性的美食 。

3. 目前，锅包肉在东北三省发展出不同风味流派：
4. 
5. 黑龙江（哈尔滨）传统派‌：坚持老式做法，使用白醋与白糖调汁，不加番茄酱，突出醋香与酸甜平衡，最具原始风味 。
6. 辽宁改良派‌：加入番茄酱调色增味，酱汁偏红亮，口感更柔和，适合大众口味 。
7. 吉林创新派‌：尝试水果融合，如菠萝锅包肉，增添果香解腻，受到年轻群体喜爱 。

## 🍖 家庭版锅包肉详细做法（2-3人份）
✅ ‌核心食材清单 ##
主料‌：猪里脊肉 ‌300g‌（选通脊更嫩）
挂糊‌：土豆淀粉 ‌210g‌（必须用土豆淀粉，玉米淀粉不酥脆）
辅料‌：大葱 50g、姜 30g、蒜 3-4瓣、香菜 10g、胡萝卜 10g（可选）
调味汁‌：白醋 ‌40g‌（建议9度白醋，酸香更正）、白糖 ‌40g‌、料酒 20ml、盐 8g、味精 5g、小苏打 5g、食用油 1000ml（炸制用）
小贴士‌：不喜欢猪肉可用鸡胸肉替代，但口感略柴 。

### 🔪 ‌详细制作步骤 ###‌
处理肉片‌

里脊肉逆纹切成 ‌3mm厚、硬币大小‌ 的片，用刀背轻拍松，加 ‌盐3g、料酒10ml‌ 抓匀腌10分钟去腥。
清水浸泡5分钟去除血水，捞出沥干，避免炸时溅油发黑 。
调制脆炸糊‌

土豆淀粉加清水浸泡 ‌2小时以上‌，倒掉上层清水，留下沉淀的湿淀粉。
加 ‌5g小苏打、10g食用油‌ 抓匀成“酸奶状”糊，裹在肉片上能挂住不滴落 。
将腌好的肉片放入糊中，轻抓均匀，静置5分钟让淀粉充分吸附。
两次炸制（关键！）‌

初炸定型‌：油温 ‌160℃‌（筷子插入冒小泡），一片片下锅，中小火炸至微黄浮起，捞出控油。
复炸酥脆‌：油温升至 ‌180℃‌，倒入肉片复炸 ‌30秒‌ 至金黄酥脆，立刻捞出。这一步是外酥里嫩的核心 。
炒制糖醋汁‌

锅留底油，爆香葱丝、姜丝、蒜片。
倒入 ‌白糖40g + 白醋40g + 盐5g + 清水10g‌，小火熬至冒密集大泡、汤汁浓稠。
可加 ‌半勺橙汁浓缩液‌ 提香（可选）。
快速翻拌出锅‌

倒入炸好的肉片，大火快速翻炒 ‌10秒‌，让每片都裹上汁。
撒入胡萝卜丝、香菜段，再翻两下立即关火装盘。
注意‌：动作要快，避免肉吸汁变软，影响酥脆口感 。
#### 🌟 成功关键要点总结 ####
要点	说明
淀粉选择‌	必须用‌土豆淀粉‌，炸后起泡酥脆不回软
油温控制‌	初炸160℃定型，复炸180℃增脆，避免炸糊
汁要呛锅‌	糖醋汁必须熬到浓稠起泡，才能挂住肉片
快炒出锅‌	翻拌时间不超过15秒，保持外皮酥脆
