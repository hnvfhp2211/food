起源与文化背景
回锅肉的历史可追溯至明代“油爆猪”，清代因郫县豆瓣酱的普及而定型，菜名最早见于1909年《成都通览》 。其“回锅”之名源于民间祭祀后将熟肉回炒食用的习俗，也被称为“熬锅肉”或“会锅肉” 。正宗的川西做法以成都、乐山一带为代表，强调咸鲜微辣、蒜苗清香，是家家户户的“下饭担当” 。

最正宗做法详解（川西流派）
根据MCP资源与多篇高可信度网页内容整合，以下是被广泛认可的正宗回锅肉做法：

选材讲究‌

主料‌：首选猪后臀部的“二刀肉”（坐墩肉），肥瘦比约3:7，肉质紧实不易散 。若买不到，可用五花肉替代，但需选肥瘦均匀者 。
辅料‌：‌香蒜苗‌（非宽叶“葱蒜苗”），夏秋时节最佳，清香解腻 。
调料‌：‌郫县豆瓣酱‌（剁细）、甜面酱、生抽、料酒、姜、葱、花椒 。
烹饪步骤‌

煮肉‌：带皮五花肉冷水下锅，加姜片、葱段、花椒、料酒，大火煮开撇沫，转小火煮15–25分钟至八成熟（筷子可插入无血水） 。
切肉‌：捞出晾凉，放入冰箱急冻2–3分钟，便于切出‌2mm厚的薄片‌，做到“上肥下瘦” 。
煸炒‌：
热锅不放油，直接下肉片中火煸炒，逼出猪油，炒至肉片卷曲呈“‌灯盏窝‌”状 。
拨肉至一边，下姜蒜、豆瓣酱炒出红油，再混合翻炒。
加甜面酱提味，倒入蒜苗白段翻炒，再加蒜苗叶断生。
最后调入生抽、少许糖，翻炒均匀即可出锅 。地方流派与变体
回锅肉在四川各地演化出丰富变体，体现“一菜百味”：

自贡流派‌：受盐帮菜影响，口味干香浓郁，辅料灵活，如“‌火爆灯盏窝‌”配紫芽姜、“‌酥锅魁烩肉‌”加锅盔 。
广汉连山回锅肉‌：选用大片坐墩肉，炒后竹竿晾滤油，配酥锅盔同炒，2006年入选德阳非遗 。
重庆流派‌：炒法洒脱，麻辣更重，常不加豆豉，突出江湖气关键技巧总结
火候‌：煸肉用中火，避免过焦；炒酱时火不宜大，防糊。
豆瓣酱‌：必须用‌郫县豆瓣‌，剁细后下锅，炒出红油是风味关键 。
蒜苗处理‌：杆与叶分次下锅，保证口感一致。
油量控制‌：煸出的油可倒出部分，避免过腻 。
