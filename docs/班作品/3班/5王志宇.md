# 跷脚牛肉 #
![](https://qcloud.dpfile.com/pc/a4ZOs-2a1ONe4mCMqu2n8pqVSLtAj8YSq6FvV1ACtMNlz5pEXaAjfdXiOZWKlHBN.jpg)
跷脚牛肉是四川省乐山市的传统名菜，又称跷脚儿、汤锅儿，属川菜系。起源于清朝光绪年间（1875-1908年），由乐山苏稽镇周村村民周天顺创制，最初为码头食摊，因食客常跷脚就餐得名 。以牛杂为主料，配以中草药熬制汤底，制作技艺以擅用料、巧用料为要点，兼具咸鲜与麻辣风味，具有药膳功能及"以脏补脏"的中医原理 。2007年列入乐山市市级非物质文化遗产名录，2008年再次入选，后升格为四川省省级非物质文化遗产 。苏稽镇形成跷脚牛肉特色小镇，拥有282家特色门店，全市门店超1000家，年产值达17.23亿元，并发展出预包装食品和全国加盟体系 跷脚牛肉非遗馆通过活态展陈和科技赋能展示其历史文化，成为文旅融合示范点
# 跷脚牛肉的来历 #
相传，在30年代初，老百姓民不聊生，贫病交加。当时在四川乐山有位擅长中草药，精通歧黄之术的罗老中医，怀着济世救人之心，在乐山苏稽镇河边悬锅烹药，救济过往行人。此汤不仅防病止渴，还能治一般风寒感冒、胃病、牙痛等。其间，他看到一些大户人家把牛杂（诸如肠子、牛骨、牛肚、草肚之类）扔到河里，觉得很可惜。于是，他把牛杂捡回洗净后，放在有中草药的汤锅。结果发现熬出来的汤味甚是鲜香。因为汤特香，又有防病治病的功效。所以，特意来饮者络绎不绝，堂堂爆满。其间没有席位者，有的站着，有的蹲着，当时人们站着习惯一只脚搭在桌底下的横梁上。久而久之食客们便形象的起了一个“跷脚”牛肉的别称，且流传至今

# 跷脚牛肉的做法 #
其汤底以菌子、笋子、枸杞等食材熬制，呈浅黄色并带特殊鲜香，表面浮油点缀葱花与红枣 。传统吃法虽无需真“跷脚”，但仍保留江湖菜粗犷与鲜香并重特点 。点菜时可选择牛杂和牛肉，素菜常配豌豆尖和银丝粉。食用时先品凉拌牛肉牛筋，后涮煮牛肉牛杂，蘸碟可自助调配，常见搭配包括豆腐乳、小米辣、香菜、芹菜、葱花和大头菜颗粒等，亦可加入香油或原汤
