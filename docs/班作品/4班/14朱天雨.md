# 水煮鱼片 #![](https://gips3.baidu.com/it/u=2259765468,874711550&fm=3074&app=3074&f=JPEG)![](h水煮鱼片是一道经典的川菜，麻辣鲜香，鱼肉滑嫩，家常做法可以简化一些步骤，但味道依然很棒。以下是简单易操作的家常版做法：

‌食材准备‌（2-3人份）：
‌主料‌：草鱼或黑鱼1条（约1.5斤，或直接买现成鱼片300克）
‌配菜‌：豆芽、青菜、千张（按喜好搭配，约200克）
‌调料‌：
姜片、蒜末、葱花
干辣椒1小把、花椒1勺
火锅底料50克（或豆瓣酱2勺）
生抽2勺、料酒1勺、淀粉1勺
盐、糖、胡椒粉少许
蛋清1个（可选，让鱼更嫩）
‌步骤‌：
‌处理鱼片‌：

鱼片洗净沥干，加1勺料酒、1勺淀粉、少许盐、胡椒粉、1个蛋清（可选），抓匀腌10分钟。
鱼骨切块备用（如果用了整鱼）。
‌煮配菜‌：

锅中烧开水，加少许盐和油，将豆芽、青菜等配菜焯熟，捞出铺在碗底。
‌炒底料‌：

热锅冷油，放姜片、蒜末爆香，加火锅底料（或豆瓣酱）炒出红油。
放入鱼骨翻炒（没有鱼骨可省略），加适量热水（约500ml），煮5分钟出鲜味，捞出鱼骨弃用。
‌煮鱼片‌：

汤中加生抽、糖、盐调味，转小火，‌一片片‌放入鱼片（防粘连）。
鱼片变色后（约1-2分钟）连汤倒入配菜碗中。
‌

‌泼热油‌：

在鱼片上撒干辣椒段、花椒、蒜末、葱花。
另起锅烧2勺热油（七成热），淋在调料上激香。
‌关键技巧‌：
鱼片选黑鱼或草鱼，刺少肉嫩；冷冻鱼片需彻底解冻。
鱼片下锅后不要大力搅拌，避免碎掉。
泼油是灵魂步骤，油温要高才能激发出麻辣味。
这样做的水煮鱼片麻辣过瘾，鱼肉嫩滑，配菜吸饱汤汁，超级下饭！根据口味可调整辣度，怕辣可减少干辣椒和花椒的量。
