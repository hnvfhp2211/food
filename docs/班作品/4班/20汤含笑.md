# 猪头焖子  #
![](https://t14.baidu.com/it/u=1216351158,1262342464&fm=224&app=112&f=JPEG?w=500&h=500)
 
## 起源传说   ##
相传源于清末哈尔滨道台府厨房。有厨师因劳累，烀猪头时不慎睡过头，导致猪头煮得软烂不成形。主厨郑兴文急中生智，剔除骨头，将碎肉与原汤混合后压实冷却凝固，没想到道台品尝后大加赞赏，“猪头焖子” 便由此流传开来，后来官府曾改称 “猪头糕”，民间仍沿用旧称。 
##  制作工艺  ##
处理与炖煮：猪头去毛洗净，焯水去腥；加葱、姜、八角、桂皮、花椒、酱油、盐等香料，慢火炖至肉骨分离、软烂入味；
拆骨切碎：取出猪头，剔除全部骨头，将带皮猪肉切碎或撕成小块；
压实冷却：把碎肉连同部分浓稠原汤装入模具 / 纱布袋，用重物长时间压实，低温静置让肉冻凝固；传统做法有用干豆腐皮包裹再压的版本；
切片食用：完全凝固后脱模，切成薄片，可直接吃，或蘸蒜泥、酱油、辣椒油等调味。

## 风味与特点 ##
外观呈酱褐色、切面紧实，肉与肉冻结合紧密；
口感 Q 弹软糯，猪皮带来胶质感，油脂在重压下析出，吃起来不腻；
味道咸香醇厚，带着香料的复合香气；
常温或微凉食用最佳，便于切片，风味更清
