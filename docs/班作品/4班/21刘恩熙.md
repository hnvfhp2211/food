# 麻婆豆腐 
![](https://qcloud.dpfile.com/pc/id2jhnhVVeHG-pzFaJqYxmGPHz-XpN64FC33d_S5VMovg8QBUdEQj36-eKKzqnY6.jpg)#
# 来源  
麻婆豆腐（Mapo Doufu），川菜传统名菜，始创于1862年成都万福桥陈兴盛饭铺，由店主陈刘氏（因面有麻点称陈麻婆）创制，以麻、辣、烫、香、嫩、酥、鲜、活八字特点著称，2011年其制作技艺入选四川省非物质文化遗产代表性项目。该菜以豆腐与肉末为主料，配豆瓣酱、花椒等调料，经黄埔军校学员传播至全国及海外。日本厨师陈建民1958年创办四川饭店推出改良版麻婆豆腐，后通过《四川料理之旅》普及日本，美国、意大利等地亦有本土化版本。2018年入选“中国菜”四川十大经典名菜及世界川菜大会首批经典菜肴，2021年获评“天府名菜” 。日本东京等五座海外城市设立川菜推广中心，非遗传承人参与国际推介#麻婆豆腐起源于清朝同治元年（1862年）的成都万福桥边‌，由“陈兴盛饭铺”的老板娘陈刘氏创制，因其脸上有麻点，被食客亲切称为“陈麻婆”，她烹制的豆腐也因此得名“麻婆豆腐”。
# 作用 #
这道菜最初是为了满足挑油的脚夫们对‌重口味、高热量、便宜下饭‌的菜肴需求而诞生。陈麻婆将豆腐与牛肉末、豆瓣酱、花椒等调料结合，意外创造出一道麻辣鲜香、口感丰富的豆腐菜，一经推出便广受欢迎。
# 名誉 #
清末时期‌：已被《成都通览》列为成都著名食品，文人诗作中也留下“麻婆陈氏尚传名，豆腐烘来味最精”的赞誉。
‌1956年‌：公私合营后，麻婆豆腐的制作技艺由成都市饮食公司系统传承，形成八代传承谱系。
‌1958年‌：川菜厨师陈建民在日本东京开设“四川饭店”，将麻婆豆腐引入日本，并引发热潮。
‌2011年‌：其制作技艺入选‌四川省非物质文化遗产‌名录。
‌2018年‌：被评为“中国菜”四川十大经典名菜之一，并被联合国教科文组织认定为“中国味道代表”。
麻婆豆腐不仅是一道家常菜，更成为中国饮食文化走向世界的重要符号，在日本、欧美等地均有本土化版本，如日本版会减少花椒用量、增加甜味以适应当地口味。
