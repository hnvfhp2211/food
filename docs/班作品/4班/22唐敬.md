# 火锅 #

火锅（Hot Pot），又叫锅子、暖炉、打边炉 [14]。古称“古董羹” [2]，是中国传统饮食器具与烹饪方式相结合的特色美食，以沸水或汤底涮煮食材并随煮随食 [2]，包括铜锅涮肉、麻辣火锅、鸳鸯火锅等，常用食材涵盖肉类、海鲜、蔬菜及豆制品。 [3]
火锅按质地分为铜、铝、陶、搪瓷等；按所用燃料分为炭火锅、酒精火锅、煤气火锅和电热火锅等；从造型可分为设烟筒和不设烟筒两类，规格分为大、中、小号等 [14]。在古代，火锅是一种锅、炉合体的炊具，上面是锅，下面为燃炭火的炉座，可以放入各种菜肴，随煮随吃 [15]。商周时，已经出现火锅雏形 [18]。汉代，出现一种称为染炉、染杯的小铜器，推断这就是古代单人使用的小火锅。唐宋时，火锅开始盛行 [15]。从宋代火锅起，肉片先在酒、酱、椒等调味汁中浸泡，再下锅 [16]。五代时，出现过五格火锅 [15]。元朝，流传到蒙古一带，用来煮牛羊肉 [15]。清朝年间，火锅不仅在民间盛行，而且成了一道著名的“宫廷菜” [19]。2007年，重庆市荣获中国烹饪协会授予中国火锅之都称号 [17]。现代火锅具有多样品种，用料丰富，别具风味等特点 [14]。发源于重庆的麻辣火锅，清代以牛油锅底涮烫毛肚等牛内脏为特色 [4] [8]，现代发展为川渝火锅代表。潮汕牛肉火锅以清汤煮鲜切牛肉闻名 [11]。衍生出海鲜火锅、菊花火锅、酸汤火锅等品类 [1] [10]，食用时讲究蘸料调配与烫煮顺序 [7]。
火锅文化体现围炉共食传统 [8]，已成为中华美食的一大特征 [18]，相关产业链已形成种植养殖、加工、研发的完整体系 。火锅还传播到世界各地，成为中外文化交流的桥梁。 [18]
