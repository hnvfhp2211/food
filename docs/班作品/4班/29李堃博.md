 # 九转大肠 #

九转大肠（Braised intestines in brown sauce），是山东省济南市传统名菜，属鲁菜系，以猪大肠为主料，加以葱、姜、芫荽等配料，经煮、烧等技法制作，成菜色泽红润，质地软嫩，兼具酸、甜、香、咸、辣五味 。该菜由清光绪年间济南九华楼厨师首创，因文人借用道家“九转仙丹”典故命名 。初加工工艺在《中国名菜谱》中记载有两种方法，主要区别在于清洗大肠使用盐醋或白矾，以及烧制时是否过油 。
制作时需将大肠套叠成型，经焯水、煎炸后加调料烧焖，并撒肉桂粉、砂仁粉提味，最后淋花椒油装盘。2018年9月入选“中国菜·山东十大经典名菜” 。其传统做法要求保留适量大肠原味，通过反复清洗与套叠工艺平衡口感 。
