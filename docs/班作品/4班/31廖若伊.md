# 炒面 #

中国传统面食小吃，以面条或面粉为主料，搭配鸡蛋、肉丝、蔬菜等食材翻炒制成，具有香辣咸鲜等特点。其分为炒面条与炒面粉两类：炒面条包含广州豉油皇炒面、潮汕干炒面、山东拌炒面等地域分支，兰州炒面片作为地方特色与牛肉面并列为当地代表性面食 ；炒面粉由谷物炒熟磨制而成，可直接冲调食用，北方油茶面、藏区糌粑均属此类 
常规炒面条制法包含煮面过水、配菜翻炒、调味拌炒等步骤，需掌握火候与面条熟度控制技巧 。炒面粉具有补充能量、健脾益胃等功效，适合胃肠功能较弱人群，但胃酸不足者需慎用。历史上炒面粉曾作为抗美援朝时期志愿军便携军粮，以“一把炒面一把雪”的食用方式著称 。现代烹饪中智能炒菜机已可实现标准化制作，提升出餐效率。
# 基本介绍 #
炒面条是北方的一道经典面食，口感筋道，味美鲜香 。其制作核心在于面条需煮至八成熟后过凉水，以保持筋道口感，并通过煎炒、焖制等手法形成干香风味 。作为地方特色美食，陕西长武县亭口镇的亭口炒面具有一定的知名度。其烹饪注重火候掌控，面条筋道，口味鲜香。亭口炒面在周边多个县市有经营门店。2020年11月，有亭口炒面获得“咸阳名吃”称号；2023年4月，在第三届陕西面食大会相关比赛中获得奖项。亭口炒面被列入长武县非物质文化遗产名录

# 主要特点 #
炒面条口感筋道光滑，口味鲜香醇厚，油润不腻 。成品配色鲜亮诱人，如鸡蛋黄、菜绿、西红柿红。制作工艺注重火候控制，例如亭口炒面强调精准掌控‘炝火候’
