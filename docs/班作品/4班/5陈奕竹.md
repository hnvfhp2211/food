# 做法 #


调料选用家家常见食材，不同家庭风味各异，形成独特饮食文化特征。2018年入选首批“中国菜”四川十大经典名菜，2021年获评省级天府旅游美食及“天府名菜”。2019年四川省提出建立回锅肉优质肉品供应基地，保障原料品质。蒜苗回锅肉蒜苗回锅肉是川菜系的传统菜品，起源于四川省及重庆市，以带皮五花肉或后腿二刀肉为主料，配以蒜苗、郫县豆瓣酱、甜面酱等调料烹制而成，具有鲜香微辣、色香味俱佳的特点。制作时需将猪肉整块煮至七八成熟，冷却后切厚片煸炒出油，形成"灯盏窝"状，随后加入豆瓣酱、甜面酱等调味料炒香，最后与拍破的蒜苗段快速翻炒断生。其核心工艺在于火候把控，煸炒时间决定口感软糯或干香，肉片厚度通常保持2毫米左右以确保香而不腻。传统做法强调食材处理细节：肉片需冷却定型后切制，蒜苗按头尾分段斜切以适配火候，调味时因豆瓣酱自带咸味可免去额外加盐。部分做法辅以豆豉、青椒等配料增强风味层次，成品常被定位为家常菜或宴客菜。手撕包菜回锅肉手撕包菜回锅肉是一道以包菜和五花肉为主料的中式家常菜，属咸鲜口味的中餐范畴。该菜品以手撕包菜保留纤维口感为核心特征，五花肉煸出油脂后，搭配蒜、姜、红辣椒等辅料爆香，再与包菜快炒并辅以蚝油调味。制作要点包括：利用五花肉自身油脂减少额外用油量，调味时需注意蚝油含盐分需控制加盐量，包菜翻炒时间可调整口感（脆爽或软烂）。部分做法提及或以老干妈辣椒酱替代部分调味料，并提供五香粉等辣度调整方案；亦有做法使用郫县红油豆瓣酱、甜面酱等调味，并可加入豆干等配料。泡菜回锅肉泡菜回锅肉是一道以五花肉、泡菜为主料制作的川味家常菜。泡菜的加入使这道菜味道咸酸微辣，口感脆生，具有开胃下饭的特点。其做法通常为先煮后炒，将煮熟的五花肉切成薄片，与切好的泡菜一同下锅急火快炒。炒制时需注意泡菜本身带有咸味，盐应少放。川菜菜谱共42个词条1.7万阅读鱼香茄子鱼香茄子是一道川菜经典菜肴，外文名称为Eggplants
