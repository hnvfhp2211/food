# 麻婆豆腐 #

 ![](https://q1.itc.cn/images01/20241024/fa761812bc984bfeb5018dd85d15e012.jpeg)
麻婆豆腐，原名陈麻婆豆腐，是川菜传统代表菜品之一，起源于清朝同治元年（1862年）成都的“陈兴盛饭铺”，由脸上有麻点的老板娘陈麻婆创制 以卤水豆腐（或嫩豆腐）、牛肉末为核心原料，辅以郫县豆瓣酱等调料，突出麻、辣、烫、香、酥、嫩、鲜、活的味型特点 
1. 今天要跟大家分享的是一款经典传统川菜，也是我们耳熟能详的，那就是麻婆豆腐，我知道这也是好多人的最爱。这道菜的主要原料是健康又便宜的豆腐，它的特色可以用8个字来概括，麻、辣、烫、香、酥、嫩、鲜、活。如果能做到以上几点，你就得到麻婆豆腐的真传了。好了，闲话少说，马上进入麻婆豆腐的制作了，首先我们看制作这道菜需要哪些原料：卤水豆腐一块，牛肉末适量，大蒜叶适量，姜、蒜适量，盐，鸡粉适量，郫县豆瓣酱适量，豆豉(切碎)适量，红干辣椒碎适量，花椒面适量(炒好的花椒用擀面杖碾碎)，
2. 红辣椒油适量，花椒油适量，香油适量，水淀粉适量。
3. 我们先来处理一下这道菜的主要原料卤水豆腐，在选择豆腐的时候尽量不要选择内酯豆腐，因为比较容易碎，要选择卤水或者老豆腐。
4. 先把豆腐切成一块1.5公分见方的小方丁就可以了，
5. 然后把切好的豆腐放入煮开水的锅里煮一下，煮的时候放些盐，目的是为了去除一些豆腥味。另外在炒的时候豆腐不容易被拨碎，变散。大概1至2分钟的时间可以把豆腐捞起了，然后沥干水分备用。
6. 接下来要处理一下其他的调味料了。把大蒜叶切碎，蒜切末，姜切末，豆豉切碎，郫县豆瓣酱切碎。蒜末不要用压蒜器，不然会水分流失，我们用刀切就可以了。处理好这些调味料就可以开始了，锅里放油先煸炒牛肉末至变色，然后放入姜蒜末、豆豉煸炒出香味，再把郫县豆瓣酱煸炒出红油，再放辣椒面，辣椒面不要选择太细的辣椒粉，要选择有粗度的辣椒面，然后一起煸炒出红油，差不多炒至所有调味料都已经均匀的溶在一起了而且香味非常浓郁的时候，就可以把焯过水的豆腐放进去了，
7. 加点清水煮一会让豆腐慢慢入味，然后依次加点花椒油、红辣椒油、芝麻香油，最后就是关键的步骤勾芡，这个芡汁不用特别浓，关键是要分两步完成勾芡，第一遍上薄薄的芡汁，包裹住每一块豆腐，然后再来一次勾芡，第二次芡汁是让整个的汤汁变得浓郁，收紧。最后放入鸡粉，
8. 再把大蒜叶碎撒上就可以关火出锅装盘，撒些花椒面即可享用。真的太好吃啦！
