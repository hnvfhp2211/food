# 鱼香肉丝的做法 #
鱼香肉丝是一道经典的川菜家常菜肴，以‌猪里脊肉‌为主料，搭配木耳、胡萝卜等辅料，口味‌咸甜酸辣‌兼备。以下是详细的家常做法及关键技巧。‌‌
食材准备
‌主料‌：猪里脊肉或瘦肉 250g-300g，切丝备用。‌
‌辅料‌：水发木耳、胡萝卜、青椒或笋丝，均切丝。‌‌‌
‌调料‌：郫县豆瓣酱或泡红辣椒、葱姜蒜末、醋、白糖、生抽、淀粉、料酒。‌‌‌
制作步骤
‌腌制肉丝‌：肉丝加少许盐、料酒、生抽、淀粉抓匀，可加蛋清或食用油锁水，腌制 10 分钟。‌‌
‌调制鱼香汁‌：碗中加入醋、白糖、生抽、淀粉及少量水搅拌均匀，糖醋比例约为 1:1 至 1.5:1。‌‌‌
‌滑炒肉丝‌：热锅凉油，下肉丝滑炒至变色发白，盛出备用。‌‌‌‌
‌炒制配菜‌：锅中留底油，炒香豆瓣酱或泡椒出红油，加入葱姜蒜末爆香，放入配菜丝翻炒断生。‌‌
‌合炒出锅‌：倒入肉丝，淋入鱼香汁，大火快速翻炒均匀，待汤汁浓稠裹住食材即可出锅。
# 历史与起源 #

编辑
鱼香肉丝起源于清末民初的四川民间。其独特的“鱼香”味型源于模仿民间烹鱼的调料和手法，因而得名。相传其首创者为清宣统三年（1911年）的四川厨师。
据考证，鱼香肉丝最早在重庆出现并得以光大。1938年，抗战时期迁至重庆出版的《南京晚报》首次提及“鱼香味”系用烹鱼的各种调合香料。1947年，重庆的《新华日报》详细记载了鱼香肉丝的菜谱。
鱼香肉丝的核心调味料也经历了演变。早期的菜谱（如《俞氏空中烹饪》）仅使用糖、醋、酱油、辣椒、姜、葱等调料。而使用泡姜、泡海椒（泡辣椒）的方法，是在上世纪60年代前后才出现的，并且首现于重庆。
传统的鱼香肉丝讲究“全肉丝”，即主料仅为肉丝，不加其他配菜（“俏头”）。随着时间推移，现代常见的做法中开始加入木耳丝、冬笋丝（或玉兰片丝）、青笋丝等配菜。一些老菜谱中记载了加入豌豆尖、莴苣笋嫩叶等配菜的版本。鱼香肉丝逐渐形成了成都（蓉派，常加配菜）与重庆（渝派，更接近传统）两种主要风味。
进入21世纪，鱼香肉丝仍在不断创新。重庆已有餐馆研发出以鱼肉为主料的“鱼肉·鱼香肉丝”，并取得了国家专利。
# 文化与影响 #
鱼香肉丝是川菜的代表性菜品，在四川美食中热度常年居高。这道菜因其酸甜可口的风味，常被选作下饭菜，同时也承载着年年有余、和和美美的美好寓意。其“鱼香味”的广泛应用及各地家庭对其配菜、口味的灵活调整，体现了四川饮食文化中的包容与创新精神。鱼香肉丝作为中华美食的代表之一，已广泛传播至海外。其英文译名“Fish-flavored Shredded Pork”曾因“鱼香”无鱼而引发误解，成为跨文化传播中的一个现象。有美国麻省理工大学的教授曾品尝过这道菜。此外，鱼香肉丝及其所代表的“鱼香味”型，已成为向世界展示中国饮食文化独特魅力的重要窗口。在2022年的中国国际进口博览会上，已有企业展出了鱼香肉丝酱等调味酱产品。‌‌‌
