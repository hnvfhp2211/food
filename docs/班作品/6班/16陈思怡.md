##  ![](https://img2.baidu.com/it/u=3755039853,1767523765&fm=253&fmt=auto&app=138&f=JPEG?w=800&h=800) <span style="font-size:32px">来源</span>
 鱼香茄子起源于四川民间，其“鱼香”并非来自鱼肉，而是源于烧鱼后剩余调料的巧妙再利用‌。相传清朝末年，四川一户擅长烹鱼的家庭主妇将烧鱼剩下的葱、姜、蒜、醋、糖等调料用来炒茄子，家人品尝后觉得味道似鱼香，由此得名。另一种说法提到川南渔民用腌鱼调料烹制茄子，形成早期雏形。

<span style="font-size:32px">地位</span>

这道菜最初在餐馆菜单上曾被称为“‌余香茄子‌”，既形容其味余韵悠长，也与“鱼香”谐音。直到上世纪70年代后才逐渐统一为“鱼香茄子”。1975年，《中国菜谱》（四川）正式收录此菜，标志着其作为川菜经典的地位得到确认。

“鱼香”是一种独特的川菜味型，以泡红辣椒、姜、蒜、糖、醋调和出咸、酸、甜、辣、鲜、香六味融合的复合风味，后广泛应用于茄子、肉丝等食材，成就了鱼香系列名菜。

<span style="font-size:32px">准备食材</span>

‌紫皮长茄子 2根、猪肉末 20g（可选）
调料：郫县豆瓣酱 1大勺、蒜末、姜末、葱花、小米辣（按口味）
鱼香汁配方（万能比例）：‌5勺清水 + 4勺醋 + 3勺生抽 + 2勺糖 + 1勺淀粉‌，搅拌均匀备用
‌处理茄子‌

茄子切成长条，加半勺盐抓匀腌制5–10分钟，挤出水分后冲洗一遍并沥干，可减少吸油。
可选：裹一层薄淀粉，炸或煎时更酥脆且少吸油。
‌‌
<span style="font-size:32px">步骤</span>

热锅凉油，放入肉末炒至变色，加入豆瓣酱、蒜末、姜末小火炒出红油。
下茄子翻炒软，倒入调好的鱼香汁，大火快速收汁。
出锅前加入青红椒丝、葱花，翻炒均匀即可。
