![](https://qcloud.dpfile.com/pc/FO_iUsq6T0zS9NOUPJWxZ9dSl8ai3kdngxsKjZf_G3kvg8QBUdEQj36-eKKzqnY6.jpg)

糖醋排骨是一道经典的中餐家常菜，以酸甜口味、色泽红亮、肉质鲜嫩著称，适合家庭制作且广受欢迎‌。

这道菜起源于浙江，是浙菜系的代表之一，后在沪菜、川菜、淮扬菜等多个菜系中发展出不同风味。主料为猪肋排或子排，通过焯水去腥、炒糖色上色、糖醋汁焖煮、收汁成菜等步骤完成。其核心在于糖与醋的黄金比例，常见有“1:2:3:4”（料酒1勺、酱油2勺、糖3勺、醋4勺）或“12345”（1酒2抽3醋4糖5水）等口诀，帮助新手掌握调味平衡 。

不同地区的做法略有差异：

‌<span style="font-size:24px">泸菜版

偏甜，常用番茄酱调色增味，口感清爽，适合家庭快炒 。
‌浙菜版‌：用料精细，讲究色香味俱全，常使用冰糖炒糖色，米醋调味，追求软烂入味 。

‌<span style="font-size:24px">川菜</span>

重油炸透，采用“炸收”技法，成菜干香滋润，多作凉菜或开胃菜 。

‌<span style="font-size:24px">广式版</span>

不过油，加入菠萝或蒜茸，用红醋和番茄汁调味，口味清淡微酸 。

<sp;;;an style="font-size:24px">推荐家庭做法</span>

猪肋排300g、冰糖30g、香醋5ml、生抽5ml、老抽5ml、蚝油5ml、姜片2片、芝麻2g、五香粉2g。
‌焯水处理‌：排骨冷水下锅，加姜片，大火煮沸后转中火焯2–3分钟，捞出用热水清洗干净。
‌炒糖色‌：锅中放少量油，小火融化冰糖至淡黄色，迅速倒入排骨翻炒上色。
‌调味炖煮‌：加入香醋、生抽、蚝油、五香粉等调料翻炒均匀，加开水没过排骨，大火煮沸后转小火焖10–15分钟。
‌收汁装盘‌：大火收汁至浓稠，撒上芝麻即可出锅 。
⚠️ 小贴士：临出锅前再加少许醋，可提升酸香味；避免长时间炖煮，以防肉质变柴。

<span style="font-size:24px">选材与预处理</span>

美味的基础
‌排骨选择‌：首选‌猪肋排中段‌（带脆骨部位），肉质鲜嫩、肥瘦适中，啃起来更带劲 。
‌清洗去腥‌：
冷水浸泡30分钟，换水1-2次，去除血水 。
‌冷水下锅焯水‌，加姜片、葱段、1勺料酒，大火煮沸后撇净浮沫。
捞出后用‌40–50℃温水冲洗‌，避免冷水导致肉质收缩变柴 。
❗避坑要点：焯水后切勿用冷水冲洗！温差会让肉质紧缩，影响后续入味和口感。

<span style="font-size:24px">黄金比例</span>


记住这个‌“54321”口诀‌（以500g排骨为基准）：

‌5勺水‌（炒糖色用）
‌4勺冰糖‌（约60g，比白糖更透亮）
‌3勺醋‌（分两次加，提升层次）
‌2勺生抽‌（提鲜不抢味）
‌1勺料酒‌（去腥增香）
进阶建议：可加1颗话梅或几片山楂，增加果酸风味，解腻又开胃 。

<span style="font-size:24px">五步成就经典</span>

❶ 热锅煎香，锁住肉汁
锅烧热后倒少量油，放入排骨中小火煎至两面微黄。
加入姜片、葱段、八角、香叶爆香。
❗避坑要点：排骨一定要沥干水分再下锅，防止油爆伤人。

❷ 炒糖色：最关键的一步
留底油，小火下冰糖，用铲子持续搅拌。
观察状态：糖粒→白泡→琥珀色（像蜂蜜色）→立即下排骨翻炒上色。
‌宁浅勿深‌：一旦变深红或冒烟，糖色会发苦 。
❸ 融合调味，炖煮入味
排骨裹上糖色后，沿锅边淋入‌第一份醋‌（总量的2/3），激发香气。
加入生抽、老抽、料酒，翻炒均匀。
倒入‌热水‌（没过排骨），大火烧开后转小火，加盖焖煮‌20–30分钟‌。
判断标准：筷子能轻松插入肉中，骨头周围微脱骨。
❗避坑要点：必须加热水！冷水会让肉质变柴。

❹ 收汁提香，画龙点睛
开大火收汁，汤汁变浓稠时，沿锅边‌淋入剩余1/3醋‌（“锅边醋”技法），瞬间激发出清新酸香 。
持续翻炒至汤汁浓稠、均匀裹在排骨上，呈“琉璃状”。
❗避坑要点：收汁时不盖锅盖，利于水分蒸发，酱汁更易挂壁。

❺ 装盘点睛
盛出后撒上‌白芝麻‌或点缀葱花，提升颜值与香气。
<span style="font-size:24px">常见问题与解决方案</span>


排骨发柴	焯水后用冷水洗、炖煮时间不足	用温水洗，炖够20分钟以上
酱汁太稀	收汁不彻底或未勾芡	大火敞锅收汁；可加1勺水淀粉（5g淀粉+15ml水）勾薄芡
味道过甜	醋加得太早或量不够	出锅前补醋；掌握“3:4”糖醋比
颜色发黑	糖色炒过头或用铁锅收汁	控制火候；建议使用不粘锅或不锈钢锅

<span style="font-size:24px">进阶玩法</span>（可选）

‌梅子版‌：收汁前加5颗话梅，酸甜中带果香，解腻满分 。
‌橙香版‌：用鲜橙汁替代部分醋，果酸柔和，适合老人小孩。
‌空气炸锅版‌：排骨腌制后200℃炸15分钟，再裹酱汁，减少用油
