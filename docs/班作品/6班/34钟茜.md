# </span><center>鱼香肉丝</center>

**鱼香肉丝是一道经典的川菜**，以咸、酸、甜、辣、鲜五味调和著称，虽名含“鱼香”，实则无鱼，而是用泡椒、姜、蒜、糖、醋等调料模拟出烹鱼的风味‌ 。

## <span style="font-family: 华文琥珀; font-size: 20px; color: red;">简介</span>

**这道菜起源于民国时期的四川民间**，最早于1947年在重庆《新华日报》中被正式记载 。其核心在于调制一碗比例精准的“鱼香汁”，并通过旺火快炒，使肉丝嫩滑、配料爽脆，成菜色泽红亮、葱姜蒜香浓郁。
鱼香肉丝‌起源于清末民初的四川民间，是一道“吃鱼不见鱼”的智慧之作，其“鱼香”并非来自鱼肉，而是模仿四川人烹鱼时常用的泡椒、姜、葱、蒜、醋、糖等调料调出的独特复合味型 。

关于它的来历，流传最广的说法是：
过去四川一户人家爱吃鱼，烧鱼时常用姜、葱、蒜、醋、酱油等去腥提鲜。一次，女主人为不浪费剩余调料，便用这些“鱼味”配料炒了肉丝。没想到丈夫尝后大加赞赏，称其风味独特，这道“鱼香炒”由此诞生，后演变为“鱼香肉丝” 。

另一种说法认为，这道菜由民国时期川菜厨师在“泡椒肉丝”基础上创新而成，1947年首次正式记载于重庆《新华日报》 。还有说法称，抗战时期蒋介石的川菜大厨根据其口味偏好，融合江浙“荔枝口”甜酸风味，最终定型并命名“鱼香肉丝” 。

如今，鱼香肉丝已成为川菜中“鱼香味型”的代表，2018年被评为“四川十大经典名菜”，2021年入选“天府名菜” ，是川菜走向全国乃至世界的一张味觉名片。

## <span style="font-family: 华文琥珀; font-size: 20px; color: red;">主要做法与材料</span>
 

主料‌：里脊肉 200g、胡萝卜 100g、青椒 100g、木耳（干）5g
调料‌：生抽、料酒、醋、白糖、豆瓣酱、姜、葱、蒜、淀粉、蛋清
烹饪步骤：
腌肉‌：将里脊肉切丝，加入生抽5ml、料酒5ml、淀粉5g、蛋清1个、水20ml拌匀，腌制15–30分钟。
准备配菜‌：木耳泡发后切丝，胡萝卜、青椒切丝，姜蒜切末，葱切小段。
调鱼香汁‌：生抽5ml + 醋15ml + 白糖10g + 盐1g + 淀粉5g + 水20ml，搅匀备用。
滑炒肉丝‌：热锅加油15ml，下腌好的肉丝快速滑散至变白，盛出。
爆香炒配菜‌：锅中加5ml油，爆香葱姜蒜和15g豆瓣酱，加入胡萝卜丝翻炒20秒，再放入青椒和木耳翻炒2分钟。
合炒收汁‌：倒入炒好的肉丝，翻炒均匀后淋入鱼香汁，大火收汁15秒即可出锅。
小贴士：鱼香汁的糖醋比约为 ‌1:1.5‌，喜欢偏酸口可适当增加醋量 。使用郫县豆瓣酱并炒出红油，是香味浓郁的关键 。
‌鱼香肉丝的主要配料包括猪里脊肉、黑木耳、胡萝卜、青椒（或彩椒）、泡辣椒（或郫县豆瓣酱），以及葱、姜、蒜等调味辅料‌。其“鱼香”风味并不来自鱼，而是通过泡椒、葱姜蒜、糖、醋、酱油等调料调和出咸、酸、甜、辣、鲜五味交融的独特口感 。

## <span style="font-family: 华文琥珀; font-size: 20px; color: red;">核心食材</span>
‌

猪里脊肉 200–300g（切丝腌制，嫩滑关键）
干木耳 5–10g（泡发后切丝）
胡萝卜 50–100g（切丝，增色提甜）
青椒/彩椒 50–100g（保持爽脆口感）
‌灵魂调料‌：

泡红辣椒 或 郫县豆瓣酱 15–30g（炒出红油是风味关键）
葱、姜、蒜 用量为平时的两倍（形成浓郁“鱼香”底味）
醋 15–20ml 与 白糖 10–20g（糖醋比约1.1:1，平衡酸甜）
生抽、老抽、料酒、盐、水淀粉（调色、去腥、勾芡）
‌腌肉小贴士‌：

肉丝用蛋清 + 淀粉 + 料酒抓匀，再封油腌制10–30分钟，可使口感更滑嫩
