                                                   # 糖醋排骨 #
经典制作流程
‌食材处理‌：选用‌500 克猪肋排‌，洗净后冷水下锅，加入‌姜片、料酒‌焯水煮沸，撇去浮沫后捞出用温水冲洗干净 。‌
1百科
‌上色煎炸‌：
‌油炸版‌：锅中油烧至七成热，放入沥干水分的排骨炸至‌金黄酥脆‌，捞出控油 。‌
1百科
‌煎制版‌：少油下锅，将排骨煎至两面焦，骨头酥脆即可 。‌
1百科
‌炒糖调色‌：锅中留底油，放入‌冰糖‌小火慢熬至融化呈‌深褐色或枣红色‌，倒入排骨快速翻炒均匀裹上糖色 。‌‌
2
‌焖煮入味‌：加入‌开水‌没过排骨，调入‌生抽、老抽、香醋‌，放入葱姜，大火烧开后转‌小火焖煮 30-40 分钟‌至肉质软烂 。‌‌
3
‌收汁出锅‌：转‌大火收汁‌，待汤汁浓稠包裹住排骨时，沿锅边淋入少许‌香醋‌增香，撒上‌白芝麻、葱花‌即可装盘 。‌
1百科
调味比例与技巧
‌万能糖醋汁‌：
‌12345 口诀‌：1 勺料酒、2 勺酱油、3 勺糖、4 勺醋、5 勺清水，适合新手掌握酸甜度 。‌‌
3
‌经典比例‌：水、醋、糖、老抽、料酒按‌5:4:3:2:1‌调配，色泽更红亮 。‌
‌去腥关键点‌：排骨需用‌温水浸泡‌半小时出血水，焯水时‌冷水下锅‌并加料酒葱姜，煮好后用‌温水清洗‌避免肉质收缩 。‌‌
5
‌炒糖色注意‌：必须用‌小火‌慢熬，观察糖液颜色变化，避免炒焦发苦；若掌握不好可用‌老抽‌代替上色 。‌
‌加醋时机‌：醋易挥发，建议‌分两次加入‌，焖煮时加一次入味，出锅前加一次增酸香 。‌
1百科
免油炸健康做法
‌简化步骤‌：排骨焯水洗净后，直接入锅炒香葱姜，无需油炸步骤 。‌
‌炖煮方式‌：加入调味料和适量热水，‌炖煮至软烂‌后大火收浓汤汁，口感更软嫩且不油腻 。‌
