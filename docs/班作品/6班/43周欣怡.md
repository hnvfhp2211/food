# 可乐鸡翅 #
可乐鸡翅的介绍
‌可乐鸡翅是一道甜咸适中、色泽红亮、肉质嫩滑的国民级家常菜，以鸡翅和可乐为主要食材，做法简单却风味独特，深受各年龄段喜爱‌。

这道菜融合了中式烹饪与西式饮料的创意，通过可乐中的糖分在加热过程中发生焦糖化反应，赋予鸡翅诱人的红亮色泽和微甜香气 。其口感外皮微焦、内里多汁，酱汁浓郁，常被形容为“拌饭神器” 。鸡翅经过焯水或煎制锁住肉汁后，再与可乐及酱油等调料小火慢炖，最后收汁而成，成品骨肉易脱、香气扑鼻 。

## 可乐鸡翅的做法 ##

食材‌：鸡翅中 10～12 只、可乐 500ml、生姜 2 片、葱段、料酒 20ml、生抽 15g、老抽 3g、盐 2g、冰糖或白糖 10g
难度‌：★★★（中等偏低，操作简单）
耗时‌：约30分钟
🔧 ‌详细步骤‌
焯水去腥‌
鸡翅冷水下锅，加姜片、料酒，大火煮开2分钟，撇去浮沫后捞出，用刀在两面各划两刀，更易入味 。

煎至金黄‌
锅中放少量油，小火将鸡翅煎至两面金黄，逼出多余油脂，口感更香而不腻 。

调味炖煮‌
加入姜片、葱段、八角（可选）炒香，倒入‌可乐没过鸡翅‌，加生抽、老抽、盐、糖调味。大火烧开后撇去浮沫 。

小火慢炖‌
转中小火炖10～15分钟，让鸡翅充分吸收可乐的甜香和酱汁的咸鲜 。

大火收汁‌
待汤汁变浓稠、出现“挂丝”状态时，转大火收汁，不断翻动防止糊锅，让酱汁均匀包裹鸡翅 。

装盘点缀‌
撒上葱花或白芝麻，色泽红亮，香气扑鼻，‌配米饭绝佳‌！

💡 ‌好吃的关键小贴士‌
可乐选普通款‌：百事或可口可乐均可，避免无糖可乐，否则缺少焦糖风味 。
不放老抽也上色‌：可乐本身含焦糖，加上炒糖色或生抽，颜色足够深 。
配菜可加‌：土豆、胡萝卜、玉米等蔬菜同炖，吸饱汤汁更下饭 。
空气炸锅版‌：鸡翅腌制后空气炸锅180℃烤15分钟，再加可乐焖煮，更少油健康 。
